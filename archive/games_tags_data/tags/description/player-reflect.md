# player-reflect

**カテゴリ**: player | **優先度**: low | **代表例**: VH WALLS

## 概要

| 項目     | 内容                                                                                          |
| :------- | :-------------------------------------------------------------------------------------------- |
| 要約     | プレイヤーキャラクターが壁や物体に衝突すると反射する                                          |
| 詳細     | プレイヤーが壁や障害物に当たると、物理的に反射して方向が変わる。反射の軌道を予測しながら、適切な位置や角度を調整する技術が求められる。反射は制約であると同時に、戦略的に利用できる要素でもある |
| リスク   | 反射の軌道を読み誤ると、危険な位置に移動してしまう。連続反射で制御を失う可能性がある          |
| リワード | 反射パターンを習熟すると、反射を利用して有利な位置に移動したり、スコアを稼いだりできる        |

## 実装例

```js
// BAMBOO より抜粋
// 硬い竹に当たると反射して加速
if ((c.a || c.b) && !input.isPressed) {
  if (b.height > 25) {
    play("hit");
    b.speed *= 0.6;
    b.height *= 0.7;
    avx++;
    vx *= -1; // 反射による方向反転
    particle(b.x, 87, 9, difficulty * (1 + avx) * 0.5, vx > 0 ? 0 : PI, 0.4);
  }
}
```

## 使用例ゲーム

このタグを使用しているゲーム:

- BAMBOO (`games/description/bamboo.md`) - 成長した竹に当たると反射して加速、タイミングを見て刈り取り
- LINE B (`games/description/line_b.md`) - 線の両端が壁で独立に反射、複雑な軌道で拡大する円を捕獲
- TWHOLS (`games/description/twhols.md`) - 斜め移動するボックスが壁で反射、穴を通り抜けて生存
- REFBALS (`games/description/refbals.md`) - 複数のボールが床で反射して跳ね返り続ける
