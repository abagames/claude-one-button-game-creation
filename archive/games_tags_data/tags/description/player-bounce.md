# player-bounce

**カテゴリ**: player | **優先度**: minimal | **代表例**: HOPPING P

## 概要

| 項目     | 内容                                                                                          |
| :------- | :-------------------------------------------------------------------------------------------- |
| 要約     | プレイヤーキャラクターが絶えずバウンド（跳躍）し続ける                                        |
| 詳細     | プレイヤーが地面に着地すると自動的にホッピングする。垂直移動を利用して障害物を避けたりアイテムを収集したりする。跳躍のタイミングや軌道を予測しながら、方向転換などの操作を行う必要がある |
| リスク   | 跳躍中は軌道を変更しにくく、敵や障害物との衝突リスクがある。着地位置の予測が難しい            |
| リワード | 跳躍パターンを習熟すると、空中での方向転換や着地タイミングを利用した効率的なプレイが可能      |

## 実装例

```js
// HOPPING P より抜粋
// 地面着地時に自動ホッピング
h.pos.add(h.vel);
h.vel.y += h.grv * difficulty * difficulty;
if (h.pos.y > 90) {
  h.pos.y = 90;
  h.hopTicks = 9; // 着地後にホッピング開始
}

// ホッピング発動
if (h.hopTicks <= 0) {
  play("jump");
  particle(h.pos, 9, -h.hop.y, -PI / 2, PI / 2);
  h.vel.set(h.hop.x * h.vx * difficulty, h.hop.y * difficulty);
}
```
