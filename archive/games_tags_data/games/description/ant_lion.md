# ANT LION (ant_lion)

**タグ**: field-gravity, field-rotate, on_holding-move

## 1. コア・メカニクス

| 項目 | 内容 |
|:---|:---|
| 入力 | ホールドで時計回りに歩行 / 離すと反時計回り |
| 挙動 | 中央のアリジゴクに引き込まれる蟻を操作し、回転する青い棒に掴まって脱出を試みる |
| 終了条件 | 中央の赤い「<>」マークに接触 |
| スコア | 青い棒に掴まるとmultiplier+2でスコア獲得。棒が中心に消えるとmultiplier-1 |
| 難易度 | 時間経過で棒の回転速度と中心への吸引速度が増加 |

## 2. オブジェクト仕様

| 名前 | 形状 | 動作 | 衝突時 |
|:---|:---|:---|:---|
| ant | bar(3,3-4)+脚 | 中心に向かって螺旋移動、ホールドで回転方向変更 | - |
| bars | bar(10-30,3) | 回転しながら中心に吸い込まれる | 掴まってスコア獲得 |
| antlion | text"<>" | 中央で点滅 | 接触で終了 |
| sand_ring | arc | 砂が渦を巻く視覚効果 | - |

## 3. デザインガイド分析

| 原則 | 評価 |
|:---|:---|
| (1) シンプルさと直感性 | 円形フィールドと渦巻き砂紋で重力方向を直感的に表現。蟻と棒のみのシンプル構成 |
| (2) 視覚的フィードバック | 中央の「<>」の点滅で危険を警告。青い棒への接触で音と視覚で成功を明示 |
| (3) リスク・リワード | 棒に長く掴まるほどmultiplierが維持され高得点だが、中心への接近リスクも増す |
| (4) 斬新なメカニクス | アリジゴクの重力をゲーム化。回転フィールドでの二択操作が独特の緊張感を生む |

## 4. タグ詳細

| タグ | 役割 | 戦略的意義 |
|:---|:---|:---|
| field-gravity | 中心への求心力が常に働く | プレイヤーは常に引き込まれる運命と戦い続ける緊張感 |
| field-rotate | フィールド全体が回転する | 回転方向を読み、棒への接触タイミングを計る必要がある |
| on_holding-move | ホールドで移動方向を制御 | 押す/離すの二択で回転方向を選び、最適ルートを探る |

## 5. 変奏ヒント

| 手法 | アイデア |
|:---|:---|
| Substitute (代用) | 重力を磁力に変え、N極S極で引力反発を切り替え |
| Combine (結合) | 複数の蟻を同時操作し、協力して棒を掴む |
| Modify (修正) | 棒に色を付け、同色に掴まるとボーナス、異色でペナルティ |
