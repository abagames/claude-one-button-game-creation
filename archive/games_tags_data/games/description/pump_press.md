# PUMP PRESS (pump_press)

**タグ**: #weapon-disablement, #field-1D, #rule-chain, #on_pressed-shoot, #field-auto_scroll

## 1. コア・メカニクス

| 項目 | 内容 |
|:---|:---|
| 入力 | タップで射撃、長押しで加速 |
| 挙動 | 横スクロールで進み、敵を撃って枠を太くし、ロケットで敵を押して左端で爆発させる |
| 終了条件 | 敵がプレイヤーの宇宙船に衝突 |
| スコア | ロケットで押した敵の数をn個とすると、(1+n)×n÷2点（連鎖ボーナス） |
| 難易度 | スクロール速度と敵の出現頻度・移動速度が上昇 |

## 2. オブジェクト仕様

| 名前 | 形状 | 動作 | 衝突時 |
|:---|:---|:---|:---|
| ship | char("a") | 自動で右に移動、長押しで加速 | 敵と衝突で終了 |
| shot | char("b") | 射撃時に前方へ発射、射程60で消滅 | 敵を膨張、ロケット発動 |
| rocket | char("c") | 横スクロール、弾が当たると左へ加速 | 敵を無力化して左へ押す |
| enemy | rect枠（拡大） | プレイヤーに向かって移動、弾で枠が太くなる | 3回被弾で破壊、ロケットで無力化 |

## 3. デザインガイド分析

| 原則 | 評価 |
|:---|:---|
| (1) シンプルさと直感性 | 宇宙船、弾、ロケット、敵という明快な要素。赤い枠と黄色の中心線で敵の状態が視覚的に分かる。 |
| (2) 視覚的フィードバック | 被弾で敵の枠が段階的に太くなり、ロケットで敵が押されて移動するアニメーションが明確。爆発時のパーティクル効果。 |
| (3) リスク・リワード | 多くの敵をロケットで一度に押すほど高得点だが、敵の数が多いとリスク増。加速で前進するか安全を取るかの選択。 |
| (4) 斬新なメカニクス | 敵を「無力化」して左端で爆発させる二段階攻撃。射撃とロケット発動の使い分けが独特。連鎖数の二次関数的スコアリング。 |

## 4. タグ詳細

| タグ | 役割 | 戦略的意義 |
|:---|:---|:---|
| #weapon-disablement | ロケットで敵を無力化し、押して移動させる | 破壊ではなく無力化という戦略的選択肢を提供し、連鎖によるスコアリングを実現 |
| #field-1D | 横スクロールの1次元フィールド | 敵の位置とタイミングの判断に集中できる |
| #rule-chain | ロケットで複数の敵を連鎖的に無力化 | 多数の敵を一度に処理する戦略性と高得点の可能性を生む |
| #on_pressed-shoot | タップで射撃、長押しで加速 | 単一ボタンで攻撃と移動速度を制御し、戦略的な使い分けを促す |
| #field-auto_scroll | 自動で右方向にスクロール | 継続的なプレッシャーと判断の連続を生み出す |

## 5. 変奏ヒント

| 手法 | アイデア |
|:---|:---|
| Substitute (代用) | ロケットを磁石や引力場に変更し、敵を引き寄せる演出に |
| Combine (結合) | 複数のロケットを同時に制御し、異なる方向の敵を処理 |
| Modify (修正) | 長押し時間でロケットのサイズや威力が変化し、より多くの敵を無力化可能に |
