# MORTAR (mortar)

**タグ**: #player-limited_vision, #weapon-explosion, #on_holding-adjust_angle, #on_released-throw

## 1. コア・メカニクス

| 項目 | 内容 |
|:---|:---|
| 入力 | 長押し（照準距離調整）/ 離す（発射） |
| 挙動 | 左右に移動する迫撃砲で、上空から落下する敵を撃墜。Hold時間で射程と爆発範囲が変化 |
| 終了条件 | 敵が地上に到達するとゲームオーバー |
| スコア | 敵撃墜でmultiplier点（1, 2, 3...）、射撃開始でmultiplierリセット |
| 難易度 | 敵の落下速度と出現数が時間経過で増加 |

## 2. オブジェクト仕様

| 名前 | 形状 | 動作 | 衝突時 |
|:---|:---|:---|:---|
| mortar | char(a) | 左右に自動移動、Hold時に照準伸長 | - |
| sight | arc(2-30) | Hold時に上方に伸び、画面スクロールで追跡 | Release時に発射/爆発 |
| shot | box(width) | 照準が近い位置でRelease時に発射、細い弾 | 敵撃墜→+multiplier点 |
| explosion | arc(radius) | 照準が遠い位置でRelease時に爆発、広範囲攻撃 | 敵撃墜→+multiplier点 |
| enemy | char(b) | 上空から落下、照準中は速度減少 | 地上到達→終了 |

## 3. デザインガイド分析

| 原則 | 評価 |
|:---|:---|
| (1) シンプルさと直感性 | Hold=照準伸長、Release=発射という直感的な操作。照準の円の大きさで爆発範囲を視覚化 |
| (2) 視覚的フィードバック | Hold時に「select」音、弾発射で「laser」音、爆発で「explosion」音。敵撃墜で「powerUp」音＋得点表示。画面スクロールで照準位置を追跡 |
| (3) リスク・リワード | 長押しで広範囲爆発できるが、その間敵が接近。短押しで素早く対処できるが、狭い範囲のみ。multiplierを維持するには連続撃墜が必要 |
| (4) 斬新なメカニクス | 照準が「見えない遠方」まで伸び、画面がスクロールして追跡する独特の視点。距離に応じて弾と爆発が切り替わる二段階の攻撃システム |

## 4. タグ詳細

| タグ | 役割 | 戦略的意義 |
|:---|:---|:---|
| #player-limited_vision | 照準が画面外まで伸びると画面がスクロールし、地上の敵が見えなくなる | 遠方を狙うと近くの危険を見逃すリスク。視界と射程のトレードオフ |
| #weapon-explosion | 遠距離照準でReleaseすると広範囲の爆発が発生 | 密集した敵を一掃できるが、照準時間が必要。連鎖的な撃墜でmultiplier増加 |
| #on_holding-adjust_angle | Hold時に照準が上方に伸び、距離を調整 | Hold時間が攻撃の性質を決定。敵の配置を見て最適な距離を選ぶ判断力 |
| #on_released-throw | Release時に発射または爆発 | タイミングが全て。照準中も敵は落下し続けるため、いつReleaseするかの判断が重要 |

## 5. 変奏ヒント

| 手法 | アイデア |
|:---|:---|
| Substitute (代用) | 迫撃砲を「弓矢」に変更し、照準を横方向に伸ばす横スクロール型に |
| Combine (結合) | 複数の砲台を同時操作し、異なる位置から同時射撃する戦略性を追加 |
| Modify (修正) | 照準の伸長速度を段階的に変化させ、特定の距離で最適な爆発範囲を得られる「スイートスポット」を設ける |
