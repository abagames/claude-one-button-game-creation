# SLALOM (slalom)

**タグ**: #on_holding-rotate, #field-auto_scroll, #rule-pass_through_gap

## 1. コア・メカニクス

| 項目 | 内容 |
|:---|:---|
| 入力 | 長押しで回転 |
| 挙動 | プレイヤーは常に前進し、長押しで左右に回転。左右交互に出現する壁の隙間を通過し、壁のエッジに近づくほど高得点。壁、岩、または画面外に出るとゲームオーバー。 |
| 終了条件 | 壁、岩、または画面外に接触 |
| スコア | 壁のエッジに近づくほど高得点（最短距離に応じて） |
| 難易度 | 時間経過で前進速度が増加 |

## 2. オブジェクト仕様

| 名前 | 形状 | 動作 | 衝突時 |
|:---|:---|:---|:---|
| player | bar（青） | 常に前進、長押しで回転、パーティクル噴射 | - |
| walls | box（赤） | 左右交互に出現、隙間がある | 接触で終了 |
| rocks | box（紫） | ランダムに出現 | 接触で終了 |

## 3. デザインガイド分析

| 原則 | 評価 |
|:---|:---|
| (1) シンプルさと直感性 | 基本図形（bar, box）で構成。壁の隙間が明確で、プレイヤーの進行方向も一目瞭然。パーティクルの噴射が飛行感を演出。 |
| (2) 視覚的フィードバック | 回転時にパーティクル噴射、壁通過時に音、ゲームオーバーで爆発。壁のエッジに近づくほど高得点という明確なフィードバック。 |
| (3) リスク・リワード | 壁のエッジに近づくほど高得点だが、接触のリスクも高まる。ギリギリを狙うことで高得点を狙える戦略性。 |
| (4) 斬新なメカニクス | 常に前進しながら回転して隙間を通過する独特のメカニクス。スラロームスキーのような動きが面白い。壁のエッジまでの最短距離でスコアが決まる独特のスコアリング。 |

## 4. タグ詳細

| タグ | 役割 | 戦略的意義 |
|:---|:---|:---|
| #on_holding-rotate | 長押しで回転、回転方向は次の壁の位置に依存 | 適切なタイミングで長押しして回転し、隙間を通過する技術が求められる |
| #field-auto_scroll | 画面が自動スクロール、プレイヤーは常に前進 | スクロール速度に合わせて適切に回転する必要があり、リズム感が求められる |
| #rule-pass_through_gap | 壁の隙間を通過することが目標、エッジに近づくほど高得点 | リスクを取って壁のエッジに近づくことで高得点を狙える。ギリギリを狙う戦略性 |

## 5. 変奏ヒント

| 手法 | アイデア |
|:---|:---|
| Substitute (代用) | 回転を別の動作（ジャンプやダッシュ）に置き換え、異なる通過方法を提供 |
| Combine (結合) | 回転以外のアクション（加速やブレーキ）を追加し、より複雑な戦略を可能にする |
| Modify (修正) | 壁の隙間の幅を変化させ、難易度を調整する |
