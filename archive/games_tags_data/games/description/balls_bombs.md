# BALLS BOMBS (balls_bombs)

**タグ**: #obstacle-bounce, #on_got_item-power_up, #weapon-explosion, #rule-combo_multiplier, #field-auto_scroll, #on_holding-move

## 1. コア・メカニクス

| 項目 | 内容 |
|:---|:---|
| 入力 | ホールドで歩行加速 / タップで瞬間加速 |
| 挙動 | 跳ねるボールを避けながら歩く。爆弾アイテムを取ると、ボールに触れて爆発で一掃 |
| 終了条件 | 通常状態でボールに接触 |
| スコア | 爆発でボールを破壊。連続破壊でmultiplier+1ずつ増加、爆発終了でリセット |
| 難易度 | 時間経過でスクロール速度とボールの出現頻度増加 |

## 2. オブジェクト仕様

| 名前 | 形状 | 動作 | 衝突時 |
|:---|:---|:---|:---|
| player | char("cd") | ホールドで歩行、減衰あり | ボール接触で終了 |
| player_bomb | char("gh") | 爆弾持ち状態 | ボール接触で爆発 |
| balls | char("ab") | 地面・壁で跳ね返る物理挙動 | 通常時終了/爆発時破壊 |
| bomb | char("ef") | 定期的に出現 | 取得でpower_up |
| explosion | arc(赤) | 半円状に拡大 | 範囲内のボール破壊 |

## 3. デザインガイド分析

| 原則 | 評価 |
|:---|:---|
| (1) シンプルさと直感性 | 跳ねるボール(黒)、爆弾(赤導火線)、爆発(赤い円)の明確な色分け |
| (2) 視覚的フィードバック | 爆弾取得でキャラ変化。爆発の円が拡大する視覚演出。multiplierの数値表示 |
| (3) リスク・リワード | 爆弾持ち時はボールに接触するほど高得点だが、タイミングを誤ると全滅 |
| (4) 斬新なメカニクス | 敵が武器に変わる逆転。避けるゲームから迎え撃つゲームへの動的な変化 |

## 4. タグ詳細

| タグ | 役割 | 戦略的意義 |
|:---|:---|:---|
| obstacle-bounce | ボールが物理的に跳ね返る | 予測可能だが複雑な軌道。物理法則の理解が有利 |
| on_got_item-power_up | 爆弾取得で能力変化 | ゲームモードが防御から攻撃に転換。戦略の切り替えが必要 |
| weapon-explosion | 爆発で範囲攻撃 | タイミングを見極め、最も多くのボールを巻き込む判断力 |
| rule-combo_multiplier | 連続破壊で得点倍率上昇 | 爆発のタイミングと位置が重要。より多く集めてから爆発 |
| field-auto_scroll | 画面が自動スクロール | 常に前進を強制され、待つ戦略が取れない |
| on_holding-move | ホールドで移動速度制御 | 速度調整で位置取りを微調整 |

## 5. 変奏ヒント

| 手法 | アイデア |
|:---|:---|
| Substitute (代用) | 爆発を時限式に変え、取得後一定時間で強制爆発 |
| Combine (結合) | 複数の爆弾タイプ(範囲、威力、持続時間が異なる)を追加 |
| Modify (修正) | ボールの色で得点が変化、特定色を狙う戦略性を追加 |
