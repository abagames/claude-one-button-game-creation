# V BOMB (v_bomb)

**タグ**: #weapon-time_limit, #rule-friendly_fire, #on_pressed-turn

## 1. コア・メカニクス

| 項目 | 内容 |
|:---|:---|
| 入力 | 押す |
| 挙動 | 自動で左右に移動する船を操作し、ボタンを押すと進行方向を反転 |
| 終了条件 | 爆発または敵機に接触 |
| スコア | 爆発範囲内の敵機を倒すと得点（連続撃破でコンボ倍率増加） |
| 難易度 | 敵機の出現頻度と移動速度が増加、爆弾のカウントダウン速度も上昇 |

## 2. オブジェクト仕様

| 名前 | 形状 | 動作 | 衝突時 |
|:---|:---|:---|:---|
| ship | char(a/b) | 左右に自動移動、ボタンで反転、壁で自動反転 | - |
| enemy | char(a/b) | 上から降下し低空でホバリング、プレイヤーを追尾、定期的に爆弾投下 | 終了 / コンボ倍率で得点 |
| bomb | char(c)+text | 地面に着弾後カウントダウン（3-9秒）、0で爆発 | カウント減少 |
| explosion | box(6,height) | 縦に伸びる爆風、上方向に拡大しながら移動 | 終了 |

## 3. デザインガイド分析

| 原則 | 評価 |
|:---|:---|
| (1) シンプルさと直感性 | カウントダウンタイマーの数字表示で爆発タイミングが明確。船と敵機の色分け（黒と黒）で自機判別が可能。紫の爆弾と赤い爆風で危険度が視覚的に伝わる |
| (2) 視覚的フィードバック | カウントが減るたびに音が鳴り、爆弾のスケールが拡大して緊張感が高まる。爆発時のパーティクルと音で成功・失敗が明確 |
| (3) リスク・リワード | 敵の近くに留まると爆弾を誘導でき複数の敵を巻き込めるが、爆発に巻き込まれるリスクも増大。コンボ倍率システムが連続撃破を奨励 |
| (4) 斬新なメカニクス | 自分で起動した爆弾が敵にも自分にも危険という「friendly fire」の緊張感。時限爆弾のカウントダウンを見ながらの位置取りが戦略的 |

## 4. タグ詳細

| タグ | 役割 | 戦略的意義 |
|:---|:---|:---|
| #weapon-time_limit | 爆弾に3-9秒のカウントダウンタイマー付き | 爆発タイミングを予測し、敵を誘導して複数撃破する戦略が可能 |
| #rule-friendly_fire | 爆風が敵にも自機にも当たり判定 | 爆発範囲からの退避タイミングが重要。高リスク高リターンの位置取りが求められる |
| #on_pressed-turn | ボタンを押すと船の進行方向を反転 | 爆弾の位置に応じた細かい位置調整が可能。壁での自動反転と組み合わせてリズミカルな操作 |

## 5. 変奏ヒント

| 手法 | アイデア |
|:---|:---|
| Substitute (代用) | カウントダウン爆弾を「導火線が燃える爆弾」に置き換え、視覚的な緊張感を強化 |
| Combine (結合) | 爆弾を「押して移動させる」要素を追加し、パズル的な配置戦略を導入 |
| Modify (修正) | 敵が複数の爆弾を同時投下するパターンを追加し、爆発の連鎖反応を作り出す |
