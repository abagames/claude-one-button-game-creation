# ROLL HOLD (roll_hold)

**タグ**: #player-rotate, #on_holding-stop, #on_holding-shoot

## 1. コア・メカニクス

| 項目 | 内容 |
|:---|:---|
| 入力 | タップで回転方向切り替え、長押しで角度固定＆射撃 |
| 挙動 | 砲塔が自動回転し、長押しで角度を固定して敵を射撃、敵の弾を砲塔で防ぐ |
| 終了条件 | 敵弾がプレイヤーに命中 |
| スコア | 敵を射撃で破壊すると9点（時間経過で減少）、砲塔で破壊すると10点 |
| 難易度 | 敵の移動速度と発射頻度が上昇 |

## 2. オブジェクト仕様

| 名前 | 形状 | 動作 | 衝突時 |
|:---|:---|:---|:---|
| player | char("a") | 中央に固定 | 敵弾で終了 |
| turret | bar(2, 3) | 自動回転、長押しで停止 | - |
| shot | bar(3, 2) | 長押し中に発射、直線移動 | 敵を破壊 |
| enemy | char("b"-"d") | 左右から出現、端で発砲 | 射撃/砲塔で破壊→得点 |
| bullet | box(4, 4) | 敵がプレイヤーを狙って発射 | 砲塔で破壊→10点 |

## 3. デザインガイド分析

| 原則 | 評価 |
|:---|:---|
| (1) シンプルさと直感性 | 中央の砲台、回転する砲塔、左右の敵という明快な配置。青い砲塔、水色の弾、赤い敵弾の色分けが明確。 |
| (2) 視覚的フィードバック | 砲塔の回転と停止、射撃のパーティクル、敵弾の軌道、破壊エフェクトが分かりやすい。 |
| (3) リスク・リワード | 長押しで射撃するが、砲塔が停止し防御が弱くなる。敵を早く倒すか、確実に防御するかの選択。 |
| (4) 斬新なメカニクス | 砲塔が攻撃と防御の両方の役割を持ち、長押しで攻撃モードに切り替わる。回転方向の切り替えによる戦略的な角度調整。 |

## 4. タグ詳細

| タグ | 役割 | 戦略的意義 |
|:---|:---|:---|
| #player-rotate | 砲塔が自動で回転し続ける | 継続的な変化により、最適なタイミングを見極める技術を要求 |
| #on_holding-stop | 長押しで砲塔の回転を停止 | 攻撃のタイミングと角度を正確にコントロールする技術を提供 |
| #on_holding-shoot | 長押し中は連続射撃 | 防御（回転）と攻撃（射撃）を切り替える戦略的な選択を促す |

## 5. 変奏ヒント

| 手法 | アイデア |
|:---|:---|
| Substitute (代用) | 砲塔をレーザー砲や魔法の杖に変更し、テーマを変える |
| Combine (結合) | 複数の砲塔を異なる速度で回転させ、より複雑な防御を実現 |
| Modify (修正) | 長押し時間で砲塔のサイズが変化し、より大きな範囲を防御可能に |
