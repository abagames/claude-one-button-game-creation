# CLEAN ROBO (clean_robo)

**タグ**: rule-pass_through_gap, rule-geometry, player-rotate, on_holding-stop, on_holding-move

## 1. コア・メカニクス

| 項目 | 内容 |
|:---|:---|
| 入力 | 長押し |
| 挙動 | 三角形ロボットが左右に往復、長押しで加速＋回転停止、離すと減速＋自動回転、壁の隙間を通りゴミを取得 |
| 終了条件 | 壁が画面外（x<0 or x>199）に出る |
| スコア | ゴミ取得で倍率分の得点、倍率+10 |
| 難易度 | 時間経過で速度増加（0.2×difficulty）、回転速度増加（0.05×difficulty） |

## 2. オブジェクト仕様

| 名前 | 形状 | 動作 | 衝突時 |
|:---|:---|:---|:---|
| robot | 三角形（3本のbar） 薄黒/黒 | 左右往復（速度1-4）、離すと回転（0.05×difficulty） | - |
| wall | line 紫 | 三角形の隙間、角度と間隔（30-100）が変化、ゴミの有無で変化 | ロボット反転 |
| garbage | box(4) 黄 | 壁の中央付近、取得で得点、壁間隔-10 | +multiplier点、倍率+10 |
| leftGarbage | box(4) 赤 | 取り損ねたゴミ、壁間隔+20のペナルティ | - |

## 3. デザインガイド分析

| 原則 | 評価 |
|:---|:---|
| (1) シンプルさと直感性 | 三角形ロボットと壁の隙間という幾何学的構成、色分け（黄=ゴミ、赤=ペナルティ、紫=壁） |
| (2) 視覚的フィードバック | 回転アニメーション、倍率表示、壁間隔の変化、残されたゴミの視覚化 |
| (3) リスク・リワード | ゴミを取ると壁間隔が狭まるリスク、取らないと間隔が広がるがペナルティ、加速と回転停止のトレードオフ |
| (4) 斬新なメカニクス | 長押しで回転停止（on_holding:stop）と加速（on_holding:move）の同時実行、壁間隔の動的変化、幾何学的パズル要素 |

## 4. タグ詳細

| タグ | 役割 | 戦略的意義 |
|:---|:---|:---|
| rule-pass_through_gap | 壁の三角形の隙間を通過、角度と間隔の幾何学的制約 | タイミングと角度の精密制御、隙間通過の習熟、間隔管理の戦略性 |
| rule-geometry | 三角形の回転角度、壁の角度（±PI/7）、幾何学的衝突判定 | 角度計算による通過可能性の判断、回転タイミングの最適化 |
| player-rotate | 離すと自動回転（0.05×difficulty）、長押しで停止 | 回転制御による隙間通過、角度調整の習熟、回転速度への適応 |
| on_holding-stop | 長押しで回転停止、角度固定 | 特定角度での通過、精密制御、加速との同時実行 |
| on_holding-move | 長押しで加速（速度+0.3）、離すと減速（→1） | 速度制御による通過タイミング調整、加速リスク管理 |

## 5. 変奏ヒント

| 手法 | アイデア |
|:---|:---|
| Substitute (代用) | 三角形を円や四角に変更 / 壁の形状を変更して異なる隙間パターン |
| Combine (結合) | 複数のゴミを配置 / 壁の種類を増やして特殊効果を追加 |
| Modify (修正) | 壁間隔の変化量を調整してリスクリターンを強化 / 回転速度を可変に / ゴミの配置をより戦略的に |
