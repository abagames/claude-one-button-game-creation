# SHINY (shiny)

**タグ**: #field-weather, #field-holes, #on_holding-reverse_state, #on_holding-move

## 1. コア・メカニクス

| 項目 | 内容 |
|:---|:---|
| 入力 | 長押しで雨を降らせる |
| 挙動 | 晴れの時は人々がゆっくり歩き、雨を降らせると人々が走り出す。雨粒が地面（ブロック）を侵食し、穴を開ける。人々が穴に落ちると脱落。右端の地面は時間経過で侵食され、タップすると加速。全員が右端まで到達すればスコア獲得、全員脱落するとゲームオーバー。 |
| 終了条件 | 人々が全員脱落 |
| スコア | 人々が右端に到達すると得点（地面の残量と人数に応じて） |
| 難易度 | 時間経過で人数が増加 |

## 2. オブジェクト仕様

| 名前 | 形状 | 動作 | 衝突時 |
|:---|:---|:---|:---|
| humans | カスタムキャラ（a/b/c/d/e） | 歩く/走る、雨で走り出す | 地面がなければ落下 |
| clouds | box（light_black） | 晴れ/雨で位置が変化 | - |
| drops | line（青） | 落下し、地面を侵食 | 人に当たると走り出す / 地面を侵食 |
| grounds | box（黒/light_black） | 雨粒で侵食され、時間経過で回復 | 人を支える |

## 3. デザインガイド分析

| 原則 | 評価 |
|:---|:---|
| (1) シンプルさと直感性 | 太陽と雲で晴れ/雨が視覚的に明確。人々のキャラクターも歩く/走るアニメーションで状態が一目瞭然。地面の侵食も視覚的にわかりやすい。 |
| (2) 視覚的フィードバック | 雨の開始/終了で音、人々の到達で音、脱落で音。雨粒が地面に当たると音とエフェクト。太陽の光線アニメーションが美しい。 |
| (3) リスク・リワード | 雨を降らせると人々が走り出すが、地面も侵食される。タイミングを見計らって雨を降らせる必要がある。右端の地面が侵食されすぎると脱落のリスクが高まる。 |
| (4) 斬新なメカニクス | 天候を操作して人々の行動を制御する独特のメカニクス。地面の侵食と回復のバランスが戦略性を生む。環境を操作して間接的に目標を達成する点が面白い。 |

## 4. タグ詳細

| タグ | 役割 | 戦略的意義 |
|:---|:---|:---|
| #field-weather | 晴れ/雨の天候変化、長押しで雨、離すと晴れ | 天候を適切に切り替えて、人々の速度と地面の侵食をバランスさせる必要がある |
| #field-holes | 地面に穴が開き、人々が落下する | 穴を避けるように雨のタイミングを調整する戦略が求められる |
| #on_holding-reverse_state | 長押しで雨（走る）、離すと晴れ（歩く） | 人々の速度を制御し、適切なタイミングで到達させる技術が重要 |
| #on_holding-move | 雨で人々が走り出し、前進速度が増加 | 雨の長さを調整して、人々の移動距離を制御する必要がある |

## 5. 変奏ヒント

| 手法 | アイデア |
|:---|:---|
| Substitute (代用) | 雨を別の環境効果（風や雪など）に置き換え、異なる影響を与える |
| Combine (結合) | 複数の天候効果を組み合わせ、より複雑な戦略を可能にする |
| Modify (修正) | 地面の侵食速度や回復速度を調整し、難易度を変化させる |
