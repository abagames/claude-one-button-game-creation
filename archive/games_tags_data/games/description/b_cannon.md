# B CANNON (b_cannon)

**タグ**: weapon-auto_aiming, obstacle-split, on_pressed-turn, on_holding-shoot

## 1. コア・メカニクス

| 項目 | 内容 |
|:---|:---|
| 入力 | タップで移動方向転換 / ホールドで照準・発射 |
| 挙動 | 画面の枠に沿って移動する砲台。最も近いボールを自動照準し、ホールドで発射。ボールは当たると分裂 |
| 終了条件 | ボールが砲台に接触 |
| スコア | ボールを最小サイズまで分裂させると得点。残りボール数が多いほど高得点 |
| 難易度 | ウェーブごとにボール数+1、移動速度+0.2。最大4個同時出現 |

## 2. オブジェクト仕様

| 名前 | 形状 | 動作 | 衝突時 |
|:---|:---|:---|:---|
| player | char("ab") | 枠に沿って移動、角で90度回転 | ボール接触で終了 |
| balls | arc(9→6→4) | 壁で跳ね返る、3段階で分裂 | 弾で2つに分裂 |
| rope | bar(可変,3) | ホールド10フレームで発射 | ボールを分裂させる |
| target | char("c") | 自動照準の目標表示 | - |

## 3. デザインガイド分析

| 原則 | 評価 |
|:---|:---|
| (1) シンプルさと直感性 | 黄色いボールと黒い枠の砲台。"c"マークで照準先を明示 |
| (2) 視覚的フィードバック | ホールド中の照準表示。分裂時の音とサイズ変化。ボール数による得点変化 |
| (3) リスク・リワード | ボールを分裂させるほど数が増え危険だが、全滅時の得点も増加 |
| (4) 斬新なメカニクス | 枠に沿った移動と自動照準の組み合わせ。分裂系ゲームに位置制御を追加 |

## 4. タグ詳細

| タグ | 役割 | 戦略的意義 |
|:---|:---|:---|
| weapon-auto_aiming | 最も近いボールを自動照準 | エイム不要だが、狙うボールを選べない制約。位置取りが重要 |
| obstacle-split | ボールが攻撃で分裂 | 攻撃するほど敵が増える。いつ攻撃するかの判断が必要 |
| on_pressed-turn | タップで移動方向転換 | 枠上の位置を制御し、照準位置を間接的に調整 |
| on_holding-shoot | ホールドで発射 | 10フレームの溜めが必要。移動を止めてタイミングを計る |

## 5. 変奏ヒント

| 手法 | アイデア |
|:---|:---|
| Substitute (代用) | 自動照準を手動照準に変え、枠上での角度調整を追加 |
| Combine (結合) | 複数の砲台を同時制御し、連携攻撃 |
| Modify (修正) | ボールの色で分裂回数や速度を変え、優先順位を作る |
