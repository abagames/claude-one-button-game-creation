# SNAKE 1 (snake_1)

**タグ**: #player-follow, #on_got_item-reverse_state, #on_pressed-turn

## 1. コア・メカニクス

| 項目 | 内容 |
|:---|:---|
| 入力 | 押す |
| 挙動 | グリッド上を自動移動する蛇の頭を操作。タップで現在の回転方向（右回り/左回り）に90度旋回し、ドル記号を取得 |
| 終了条件 | 壁または自分の体に衝突 |
| スコア | ドル記号を1つ取得するごとに+1点 |
| 難易度 | 時間経過で蛇の移動速度が上昇 |

## 2. オブジェクト仕様

| 名前 | 形状 | 動作 | 衝突時 |
|:---|:---|:---|:---|
| head | text(>/v/</>^) | グリッド上を自動移動、タップで回転方向に90度旋回 | - |
| body | text(o) | 頭の移動に追従、ドル取得時は消失せず長さを維持 | ゲームオーバー |
| dollar | text($) | フィールド上にランダム配置 | +1点、体が1マス伸長、回転方向反転、新たに2個出現 |
| wall | text(#) | フィールド外周に配置 | ゲームオーバー |

## 3. デザインガイド分析

| 原則 | 評価 |
|:---|:---|
| (1) シンプルさと直感性 | スネークゲームの基本ルールを踏襲。方向キーではなくワンボタンでの旋回操作に落とし込むことで、シンプルかつ緊張感のある操作を実現 |
| (2) 視覚的フィードバック | グリッドベースの明確な配置。頭の向きが矢印で表示され、次の動きが予測可能。衝突時は白い四角とXマークで終了を明示 |
| (3) リスク・リワード | ドル取得で体が伸びるため、後半ほど自分の体が障害物となる。回転方向の反転により、次の操作が逆になる認知的負荷が戦略性を高める |
| (4) 斬新なメカニクス | ドル取得時に回転方向が反転する仕組みが独特。右回りで慣れた操作が突然左回りになり、プレイヤーに適応を強いる |

## 4. タグ詳細

| タグ | 役割 | 戦略的意義 |
|:---|:---|:---|
| #player-follow | 体（bodies）が頭の移動軌跡に追従 | スネークゲームの本質。体が長くなるほど自由な移動空間が減り、計画的なルート取りが必要 |
| #on_got_item-reverse_state | ドル取得時に回転方向（rotation）が反転 | 単調な操作に変化を与える。プレイヤーは常に現在の回転方向を意識し、取得タイミングを戦略的に選ぶ必要がある |
| #on_pressed-turn | タップで90度旋回 | ワンボタンで4方向を制御する洗練された入力方式。タイミングとリズムが攻略の鍵 |

## 5. 変奏ヒント

| 手法 | アイデア |
|:---|:---|
| Substitute (代用) | ドルの代わりに敵キャラを配置し、触れると体が縮む「マイナスアイテム」として機能させる |
| Combine (結合) | 2匹の蛇を同時操作し、互いに衝突しないよう協調させる協力プレイ要素を追加 |
| Modify (修正) | 回転方向の反転を「タップ長押しで固定」できる特殊状態を追加し、戦略の幅を広げる |
