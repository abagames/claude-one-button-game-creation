# SCRAMBIRD (scrambird)

**タグ**: #on_pressed-flap, #on_pressed-shoot, #field-auto_scroll, #field-roughness, #field-spike, #rule-combo_multiplier, #rule-time_limit

## 1. コア・メカニクス

| 項目 | 内容 |
|:---|:---|
| 入力 | タップで羽ばたきと射撃 |
| 挙動 | プレイヤーは飛行機で、タップすると上昇と射撃を同時に行う。燃料がある間は強力に上昇し、燃料が切れると弱く上昇。上下の壁と地形の凹凸をかわしながら、ミサイルを撃破してスコアを稼ぐ。タンクを撃破すると燃料を回復。 |
| 終了条件 | 壁、地形、またはミサイル/タンクに接触 |
| スコア | ミサイルを撃破で得点（マルチプライヤーあり） |
| 難易度 | 時間経過でスクロール速度と重力が増加 |

## 2. オブジェクト仕様

| 名前 | 形状 | 動作 | 衝突時 |
|:---|:---|:---|:---|
| ship | カスタムキャラ（c/d） | タップで上昇と射撃、パーティクル噴射 | - |
| walls | rect（色変化） | 上下の壁と地形、高さが変化する | 接触で終了 |
| missiles | カスタムキャラ（a） | 一定時間後に上昇 | 撃破で得点 / 接触で終了 |
| tanks | カスタムキャラ（b） | 地形の上に配置 | 撃破で燃料+10 / 接触で終了 |
| shots | カスタムキャラ（e） | 右へ直進 | 敵を撃破 |
| bombs | bar（cyan） | 放物線を描く | 敵を撃破 |

## 3. デザインガイド分析

| 原則 | 評価 |
|:---|:---|
| (1) シンプルさと直感性 | カスタムキャラクターで視覚的に明確。燃料ゲージが画面下部に大きく表示され、リソース管理が一目瞭然。壁の色が時間経過で変化し、視覚的なバリエーションを提供。 |
| (2) 視覚的フィードバック | 射撃、敵撃破、燃料回復で異なる音とパーティクル。飛行機からパーティクルが常に噴射され、飛行感が強調される。ゲームオーバーで爆発エフェクト。 |
| (3) リスク・リワード | 燃料がある間は強力に上昇できるが、燃料が切れると弱くなる。ミサイルを撃ち逃すとマルチプライヤーが減少。タンクを撃破すると燃料回復するが、接触すると即ゲームオーバー。リスクを取って燃料を確保する必要がある。 |
| (4) 斬新なメカニクス | タップで羽ばたきと射撃を同時に行う1入力2アクション。燃料管理の要素が戦略性を生む。爆弾が放物線を描き、弾とは異なる軌道で敵を狙える。 |

## 4. タグ詳細

| タグ | 役割 | 戦略的意義 |
|:---|:---|:---|
| #on_pressed-flap | タップで羽ばたき（上昇）、燃料の有無で上昇力が変わる | 燃料管理が重要。燃料がある間は強力に上昇できるが、切れると弱体化する |
| #on_pressed-shoot | タップで射撃（弾と爆弾を同時に発射） | 1回のタップで2種類の弾を発射し、異なる軌道で敵を狙える。タップのタイミングが重要 |
| #field-auto_scroll | 画面が自動スクロール、地形と敵が右から左へ移動 | スクロール速度に合わせて適切に上昇・下降する必要があり、リズム感が求められる |
| #field-roughness | 地形の高さが変化する、凹凸のある地形 | 地形の変化を予測し、適切な高度を維持する必要がある。高さの変化が戦略性を生む |
| #field-spike | 上下の壁（トゲのような障害物） | 上下に壁があり、上昇しすぎても下降しすぎても即ゲームオーバー。高度管理が重要 |
| #rule-combo_multiplier | ミサイルを連続撃破でマルチプライヤー増加、撃ち逃すと減少 | 連続撃破を狙うことで高得点を狙える。撃ち逃さないように慎重なプレイが求められる |
| #rule-time_limit | 燃料が時間制限のような役割、切れると弱体化 | 燃料を確保し続ける必要があり、タンクを撃破する戦略が重要 |

## 5. 変奏ヒント

| 手法 | アイデア |
|:---|:---|
| Substitute (代用) | 燃料を別のリソース（シールドやパワーアップ）に置き換え、異なるゲーム性を提供 |
| Combine (結合) | 射撃と羽ばたきを独立した操作に分離し、より複雑な戦略を可能にする |
| Modify (修正) | 燃料の消費速度や回復量を調整し、燃料管理の難易度を変化させる |
