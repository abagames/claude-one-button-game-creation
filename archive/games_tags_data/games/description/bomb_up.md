# BOMB UP (bomb_up)

**タグ**: #weapon-blow_off, #weapon-explosion, #obstacle-well_up, #on_pressed-shoot, #on_pressed-attack, #field-auto_scroll

## 1. コア・メカニクス

| 項目 | 内容 |
|:---|:---|
| 入力 | 1回目のタップで爆弾投擲 / 2回目のタップで起爆 |
| 挙動 | 上下から湧く赤い岩を避け、爆弾を投げて爆発で破壊。爆発の反動で自機も吹き飛ぶ |
| 終了条件 | 岩が自機に接触 |
| スコア | 爆発で岩を破壊。連続破壊でmultiplier+1ずつ増加、投擲でリセット |
| 難易度 | 時間経過で岩の出現頻度増加(8-10/difficulty)、下からの岩の速度上昇。プレイヤー位置で画面スクロール(y<40で上、y>60で下) |

## 2. オブジェクト仕様

| 名前 | 形状 | 動作 | 衝突時 |
|:---|:---|:---|:---|
| player | char("ab") | 物理演算で移動、壁で減衰反射 | 岩接触で終了 |
| bomb | char("c") | 投擲後に物理演算で移動 | 起爆で爆発 |
| explosion | arc(薄赤) | 半径25の範囲攻撃 | 岩を破壊 |
| rocks | box(5,赤) | 上下から湧き、徐々に加速 | プレイヤー接触で終了 |
| walls | box(4,薄黒) | 左右の壁 | 跳ね返り |

## 3. デザインガイド分析

| 原則 | 評価 |
|:---|:---|
| (1) シンプルさと直感性 | 黒いキャラと爆弾、赤い岩、薄赤い爆発の明確な色分け |
| (2) 視覚的フィードバック | 爆発のパーティクルと範囲表示。反動による吹き飛び。岩の軌跡パーティクル |
| (3) リスク・リワード | 爆発の反動を利用した移動。岩を多く集めてから爆破すると高得点だが危険 |
| (4) 斬新なメカニクス | 爆発が攻撃と移動の両方に作用。二段階入力で起爆タイミングを制御 |

## 4. タグ詳細

| タグ | 役割 | 戦略的意義 |
|:---|:---|:---|
| weapon-blow_off | 爆発の反動で自機が吹き飛ぶ | 攻撃が移動手段にもなる。位置と距離を計算した戦略的配置 |
| weapon-explosion | 範囲攻撃 | 複数の敵を同時破壊可能。タイミングと位置が重要 |
| obstacle-well_up | 障害物が上下から湧き出る | 上下両方への注意が必要。予測困難な緊張感 |
| on_pressed-shoot | 1回目のタップで爆弾投擲 | 投擲のタイミングと方向が重要 |
| on_pressed-attack | 2回目のタップで起爆 | 起爆タイミングが戦略の核心。溜めるか即爆破か |
| field-auto_scroll | プレイヤー位置で画面スクロール | 位置制御が重要。スクロールで岩の湧き位置が変化 |

## 5. 変奏ヒント

| 手法 | アイデア |
|:---|:---|
| Substitute (代用) | 爆発を吸引力に変え、岩を集めてから消滅させる |
| Combine (結合) | 複数の爆弾を同時に投げて連鎖爆発 |
| Modify (修正) | 爆発の大きさを溜め時間で変化させ、リスク・リワードを強調 |
