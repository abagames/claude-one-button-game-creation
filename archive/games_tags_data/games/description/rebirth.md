# REBIRTH (rebirth)

**タグ**: #on_got_item-reverse_state, #player-inverted, #field-outpost, #on_pressed-jump, #on_pressed-fall

## 1. コア・メカニクス

| 項目 | 内容 |
|:---|:---|
| 入力 | タップでジャンプ、空中でタップすると急降下 |
| 挙動 | 画面が左右2つの世界に分かれ、トラックに当たると反対の世界へ反転、ダイヤを集めて得点 |
| 終了条件 | ダイヤが画面中央に到達（取り逃がす） |
| スコア | ダイヤ取得で multiplier × 1点、連続取得で multiplier 増加 |
| 難易度 | トラックの速度とダイヤの出現間隔が増加 |

## 2. オブジェクト仕様

| 名前 | 形状 | 動作 | 衝突時 |
|:---|:---|:---|:---|
| player | char("b"-"f") | 左右の世界を走り、ジャンプ/急降下 | - |
| track | char("a") | 左右から流れてくる | 衝突で反対世界へ反転 |
| dia | char("g") | 左右から流れてくる黄色のダイヤ | 取得→得点、逃すと終了 |

## 3. デザインガイド分析

| 原則 | 評価 |
|:---|:---|
| (1) シンプルさと直感性 | 画面中央で分かれた白黒2つの世界、トラックとダイヤという明快な要素。色の反転により世界の違いが一目瞭然。 |
| (2) 視覚的フィードバック | 世界の反転時の色の変化、プレイヤーの走り/ジャンプ/急降下のアニメーション、ダイヤ取り逃がし時の赤いXマーク。 |
| (3) リスク・リワード | トラックに当たって反対世界に移動することでダイヤを取れるが、タイミングを誤ると危険。ジャンプと急降下の使い分け。 |
| (4) 斬新なメカニクス | 画面が2つの対称的な世界に分かれ、トラックで反転して行き来する独特のシステム。空中で急降下できるジャンプ。 |

## 4. タグ詳細

| タグ | 役割 | 戦略的意義 |
|:---|:---|:---|
| #on_got_item-reverse_state | トラックに当たると反対の世界へ反転 | 世界間の移動をアイテム取得に統合し、戦略的な移動を実現 |
| #player-inverted | 左右で白黒が反転した対称的な世界 | 視覚的に明確な2つの世界を提供し、プレイヤーの位置を強調 |
| #field-outpost | 画面中央で分かれた2つの世界 | 空間的な制約と戦略的な移動の必要性を生む |
| #on_pressed-jump | タップでジャンプ | トラックを避けたり、ダイヤを取るための基本操作 |
| #on_pressed-fall | 空中でタップすると急降下 | ジャンプ後の軌道を調整し、タイミングの精密なコントロールを可能にする |

## 5. 変奏ヒント

| 手法 | アイデア |
|:---|:---|
| Substitute (代用) | 白黒の世界を昼夜や天国地獄に変更し、テーマを変える |
| Combine (結合) | 3つ以上の世界に分割し、より複雑な移動戦略を要求 |
| Modify (修正) | 長押しでジャンプの高さを調整可能にし、より精密な制御を実現 |
