# RAID (raid)

**タグ**: #weapon-artillery, #on_pressed-shoot, #on_holding-thrust, #rule-chain, #field-auto_scroll

## 1. コア・メカニクス

| 項目 | 内容 |
|:---|:---|
| 入力 | 長押しで加速、離すと爆弾投下 |
| 挙動 | 飛行機が横に往復し、端で下降、爆弾で建物を破壊して連鎖的に上階も崩壊 |
| 終了条件 | 飛行機が建物に衝突 |
| スコア | 破壊した階の高さ（上の階ほど高得点、multiplier 1, 2, 3...で加算） |
| 難易度 | 下降距離が増加し、画面スクロール速度が上昇 |

## 2. オブジェクト仕様

| 名前 | 形状 | 動作 | 衝突時 |
|:---|:---|:---|:---|
| ship | char("a", "b") | 左右に往復、端で下降、長押しで加速 | 建物で終了 |
| bomb | bar(3, 3) | 放物線で落下、壁で反射、重力加速 | 建物破壊 |
| building | rect(9, 10) | 各階が緑/黄/紫/青、上方向にスクロール | 爆弾/破片で破壊→得点 |
| falling | rect(9, 10) | 破壊された階が破片として落下 | 下の階を破壊 |
| cloud | box(size) | 背景装飾、上方向にスクロール | - |

## 3. デザインガイド分析

| 原則 | 評価 |
|:---|:---|
| (1) シンプルさと直感性 | 飛行機、爆弾、建物という明快な要素。カラフルな建物の各階が視覚的に区別され、破壊の進行が分かりやすい。 |
| (2) 視覚的フィードバック | 爆弾の放物線運動、建物の連鎖崩壊、破片の落下と二次破壊のアニメーションが明確。爆発エフェクトとパーティクル。 |
| (3) リスク・リワード | 加速で下降を早めると高得点の上階を狙いやすいが、建物への衝突リスク増。爆弾投下のタイミングと位置の精密な判断。 |
| (4) 斬新なメカニクス | 建物の連鎖崩壊（爆弾で1階破壊→上の全階が落下）と破片による二次破壊の組み合わせ。上階ほど高得点という高度ベースのスコアリング。 |

## 4. タグ詳細

| タグ | 役割 | 戦略的意義 |
|:---|:---|:---|
| #weapon-artillery | 放物線で落下する爆弾による砲撃 | 飛行機の速度と位置から着弾点を予測する戦略性を生む |
| #on_pressed-shoot | 長押しを離すと爆弾投下 | 長押しによる加速と爆弾投下のタイミングを統合し、単一入力で複雑な戦略を実現 |
| #on_holding-thrust | 長押しで飛行機が加速 | 速度調整により爆弾の飛距離と着弾タイミングをコントロール |
| #rule-chain | 爆弾で建物を破壊すると上の階が全て崩壊し、破片が下の階を破壊 | 連鎖的な破壊により高得点を狙える戦略性と爽快感を生む |
| #field-auto_scroll | 建物が上方向に自動スクロール | 継続的なプレッシャーと時間制限を生み出す |

## 5. 変奏ヒント

| 手法 | アイデア |
|:---|:---|
| Substitute (代用) | 飛行機をドローンやUFOに、建物を宇宙ステーションや氷山に変更 |
| Combine (結合) | 複数の爆弾を同時投下し、異なる建物を狙う戦略性 |
| Modify (修正) | 長押し時間で爆弾のサイズや威力が変化し、より多くの階を破壊可能に |
