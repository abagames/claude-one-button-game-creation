# UP DOWN PRESS (up_down_press)

**タグ**: #player-step_on, #on_holding-thrust, #field-roughness, #field-auto_scroll

## 1. コア・メカニクス

| 項目 | 内容 |
|:---|:---|
| 入力 | タップ：ジャンプ / 長押し：加速（ジャンプ中は落下速度減少） |
| 挙動 | 起伏のある道路を走る車を操作。ジャンプして下降中に他の車を踏むとスコア獲得し、マルチプライヤーが2倍に増加 |
| 終了条件 | 画面左端に到達 |
| スコア | 車を踏むとマルチプライヤー分のスコア獲得 |
| 難易度 | 時間経過で基本速度が上昇し、他の車の出現頻度も増加 |

## 2. オブジェクト仕様

| 名前 | 形状 | 動作 | 衝突時 |
|:---|:---|:---|:---|
| my_car | box(5) + bar(5) 青/シアン | 道路の起伏に沿って走行。長押しで加速、タップでジャンプ。道路の傾斜で加減速 | - |
| other_cars | box + bar 赤/黄/緑/紫 | 道路上を様々な速度で走行。道路の傾斜で加減速。前後どちらの方向にも移動 | ジャンプ下降中に踏むとマルチプライヤー分のスコア + マルチプライヤー×2。地上で接触すると押し戻される |
| road | line 薄黒 | 起伏のある道路。上下に変動しながら自動的に左にスクロール | - |

## 3. デザインガイド分析

| 原則 | 評価 |
|:---|:---|
| (1) シンプルさと直感性 | 青の自車、カラフルな他車、黒い道路という明確な色分け。レーシングゲームの直感的な操作感 |
| (2) 視覚的フィードバック | ジャンプで「jump」音、踏みつけで「powerUp」音とパーティクル。衝突で「explosion」音と押し戻し。マリオ的な踏みつけアクションが明確 |
| (3) リスク・リワード | 車を踏むとマルチプライヤー倍増で高スコアだが、失敗すると押し戻されて画面左端に近づく危険 |
| (4) 斬新なメカニクス | レーシングゲームと踏みつけアクションの融合。道路の起伏が速度に影響し、物理的なリアリティを生む |

## 4. タグ詳細

| タグ | 役割 | 戦略的意義 |
|:---|:---|:---|
| #player-step_on | ジャンプ下降中に他の車を踏むとスコア獲得 | 踏みつけのタイミングが攻略の核心。ジャンプの高さと着地位置を計算する技術が必要 |
| #on_holding-thrust | 長押しで加速し、ジャンプ中は落下速度を減少 | 加速でスコアチャンスを増やすか、慎重に進むかの選択。ジャンプ中の滞空時間調整で踏みつけ成功率が変化 |
| #field-roughness | 道路の起伏が車の速度に影響 | 上り坂で減速、下り坂で加速する物理演算。道路の形状を読んで速度を予測する戦略性 |
| #field-auto_scroll | 道路が自動的に左にスクロールし続ける | 一定のペースで前進を強制され、遅れると画面左端に到達。テンポ感が緊張を生む |

## 5. 変奏ヒント

| 手法 | アイデア |
|:---|:---|
| Substitute (代用) | 車をキャラクターに置き換え、プラットフォーマーゲーム化。または空中戦に変更し、敵機を撃墜する要素を追加 |
| Combine (結合) | アイテム収集要素を追加。道路上にコインを配置し、踏みつけとアイテム取得の両立を要求 |
| Modify (修正) | 道路を複数レーンに分割。左右移動も可能にし、レーン選択と踏みつけの両方を管理する複雑さを加える |
