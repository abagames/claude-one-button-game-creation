# CATE P (cate_p)

**タグ**: #on_pressed-turn, #on_pressed-shoot

## 1. コア・メカニクス

| 項目 | 内容 |
|:---|:---|
| 入力 | 押す |
| 挙動 | プレイヤーが8方向のいずれかに自動移動、押すと右回転（45度）＋射撃 |
| 終了条件 | 敵に衝突、または壁外に出る |
| スコア | ショットで敵を倒すと倍率分の得点、連続撃破で倍率+1 |
| 難易度 | 時間経過でプレイヤーと敵の速度増加（0.2×difficulty、1.5×difficulty） |

## 2. オブジェクト仕様

| 名前 | 形状 | 動作 | 衝突時 |
|:---|:---|:---|:---|
| player | char("a"/"b") 黒 | 8方向に自動移動（0.2×difficulty）、壁や外枠に当たると右回転 | - |
| shot | char("c") 黒 | プレイヤーの射撃方向に進む（1.5×difficulty）、外枠で消滅 | - |
| cate | box(5,5) 緑 + box(2,2)×2 赤 | 8方向に移動、壁で反射（3方向または4方向回転）、5-9個連続出現 | +multiplier点、倍率+1 |
| wall | rect 薄青 | 上下左右の外枠（幅6px）、プレイヤーの回転トリガー | プレイヤー回転 |

## 3. デザインガイド分析

| 原則 | 評価 |
|:---|:---|
| (1) シンプルさと直感性 | 1ボタンで回転と射撃を同時実行、8方向移動のシンプルな動き、色分け（青=壁、緑=敵本体、赤=触角） |
| (2) 視覚的フィードバック | 敵の触角アニメーション（往復運動）、プレイヤーの見た目変化（射撃中は"b"）、倍率表示 |
| (3) リスク・リワード | 敵に接近して射撃するリスク、連続撃破で倍率増加、壁を利用した軌道制御の習熟 |
| (4) 斬新なメカニクス | 1ボタンで回転と射撃を同時実行、壁反射による軌道制御、敵の連続出現パターン、8方向の離散的移動 |

## 4. タグ詳細

| タグ | 役割 | 戦略的意義 |
|:---|:---|:---|
| on_pressed-turn | 押すと右回転（45度、angle+2 wrap 0-8）、壁に当たっても自動回転 | タイミングによる方向制御、壁反射との組み合わせによる複雑な軌道、45度単位の離散的制御 |
| on_pressed-shoot | 押すとショット発射、プレイヤーの現在角度で直進（1.5×difficulty） | 回転と射撃の同時実行による戦略性、連続撃破での倍率増加、ショット発射中の見た目変化 |

## 5. 変奏ヒント

| 手法 | アイデア |
|:---|:---|
| Substitute (代用) | 回転を左回転に変更 / 射撃を自動化して回転のみに集中 / 8方向を4方向に簡略化 |
| Combine (結合) | 複数のショットを同時発射（扇状） / 敵の種類を増やして異なる反射パターンを追加 |
| Modify (修正) | 壁反射で減速するペナルティを追加 / 倍率減衰システムを導入して緊張感を高める / 敵の出現パターンをより予測可能に |
