# SLASHES (slashes)

**タグ**: #rule-proximity_bonus, #on_pressed-turn

## 1. コア・メカニクス

| 項目 | 内容 |
|:---|:---|
| 入力 | 押す |
| 挙動 | タップで棒の角度を左右に反転させ、斜めに飛来する敵の棒の太い部分を斬る |
| 終了条件 | 敵の棒の細い部分に触れる |
| スコア | 太い部分を連続で斬った累積スコアを、太い部分から離れたときに獲得 |
| 難易度 | 時間経過で敵の出現頻度と移動速度が上昇 |

## 2. オブジェクト仕様

| 名前 | 形状 | 動作 | 衝突時 |
|:---|:---|:---|:---|
| player | bar(4,3) 角度可変 | 画面下部で斜め45度の角度を保持、タップで左右反転 | - |
| slash(太) | bar(幅可変,22) | 斜め45度で画面上部から出現し、下+横方向に移動 | +speed×累積、particle生成 |
| slash(細) | bar(幅-20,4) | 太い部分を斬られた後に残る細い棒として移動継続 | ゲームオーバー |

## 3. デザインガイド分析

| 原則 | 評価 |
|:---|:---|
| (1) シンプルさと直感性 | 棒を斬る動作が直感的。太い部分(light_purple)と細い部分(purple)の色と幅の違いで、「斬るべきもの」と「避けるべきもの」が明確に区別される |
| (2) 視覚的フィードバック | 斬った際のパーティクルと効果音、スコア表示のタイミング（連続斬り終了時）が気持ちよい。危険な細い棒との接触でゲームオーバー |
| (3) リスク・リワード | 連続で斬り続けることでスコアが累積し、より高得点を狙える。ただし細い棒の密度も上がりリスクが増大 |
| (4) 斬新なメカニクス | 「斬る」と「避ける」が同じオブジェクトの異なる状態として表現される。連続斬りの累積ボーナスと、それを確定させるタイミングの駆け引き |

## 4. タグ詳細

| タグ | 役割 | 戦略的意義 |
|:---|:---|:---|
| #rule-proximity_bonus | 太い部分を連続で斬り続けている間、スコアが累積される | リスクを取って斬り続けるか、安全に離れてスコアを確定させるかの判断が求められる |
| #on_pressed-turn | タップで棒の角度が左右に反転 | 飛来する敵の角度に合わせて素早く棒を反転させる操作精度が重要 |

## 5. 変奏ヒント

| 手法 | アイデア |
|:---|:---|
| Substitute (代用) | 棒を円形の障害物に変え、タップで回転方向を変える「回転斬り」ゲームにする |
| Combine (結合) | 複数の棒を同時に操作し、2本の棒で敵を挟み撃ちにする協調アクション要素を追加 |
| Modify (修正) | 敵の角度を45度以外にもバリエーションを持たせ、プレイヤーの角度も3段階以上に変化させる |
