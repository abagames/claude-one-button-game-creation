# S SHAKE (s_shake)

**タグ**: #on_pressed-change_field, #field-outpost, #field-auto_scroll

## 1. コア・メカニクス

| 項目 | 内容 |
|:---|:---|
| 入力 | タップで地形を振動 |
| 挙動 | 波状の地形が自動スクロールし、タップすると地形の高低差が3倍に拡大（振動）。敵は地形の上を歩き、振動で跳ね上がる。敵を画面上部に跳ね上げると得点、左端に到達するとゲームオーバー。 |
| 終了条件 | 敵が左端（拠点）に到達 |
| スコア | 敵を画面上部に跳ね上げると得点（マルチプライヤーあり） |
| 難易度 | 時間経過で敵の速度と重力が増加 |

## 2. オブジェクト仕様

| 名前 | 形状 | 動作 | 衝突時 |
|:---|:---|:---|:---|
| grounds | line | 波状に動き、右から左へスクロール、タップで高低差3倍 | - |
| enemies | カスタムキャラ（a/b） | 地形の上を歩く、振動で跳ね上がる | 画面上部に到達で得点 / 左端到達で終了 |

## 3. デザインガイド分析

| 原則 | 評価 |
|:---|:---|
| (1) シンプルさと直感性 | 波状の地形が視覚的に明確で、敵が地形の上を歩く様子が一目瞭然。タップで地形が拡大する様子も直感的。 |
| (2) 視覚的フィードバック | タップで地形が即座に拡大し、敵が跳ね上がるアニメーション。敵を跳ね上げると得点表示が画面上部に出現。地形の波の動きも視覚的に美しい。 |
| (3) リスク・リワード | タップで地形を振動させると敵を跳ね上げられるが、マルチプライヤーがリセットされる。連続で跳ね上げるとマルチプライヤーが増加するため、タイミングを見計らう戦略性が求められる。 |
| (4) 斬新なメカニクス | 地形を振動させて敵を跳ね上げるという独特のメカニクス。地形の高低差を利用したゲームプレイが斬新。敵を直接攻撃せず、環境を操作する点が面白い。 |

## 4. タグ詳細

| タグ | 役割 | 戦略的意義 |
|:---|:---|:---|
| #on_pressed-change_field | タップで地形の高低差を3倍に拡大（振動）、敵を跳ね上げる | 適切なタイミングで振動させることで、複数の敵を同時に跳ね上げられる。地形の波の位置を把握する必要がある |
| #field-outpost | 左端が拠点で、敵が到達するとゲームオーバー | 拠点防衛の緊張感が常にあり、敵を確実に跳ね上げる必要がある。タイミングを逃すと即ゲームオーバー |
| #field-auto_scroll | 地形が右から左へ自動スクロール、敵も地形に乗って移動 | スクロールによって地形の波の位置が常に変化し、最適なタイミングを見極める判断力が求められる |

## 5. 変奏ヒント

| 手法 | アイデア |
|:---|:---|
| Substitute (代用) | 地形の振動を重力反転に置き換え、敵が上下に動く |
| Combine (結合) | 敵を跳ね上げる以外のアクション（射撃やシールドなど）を追加し、複数の対処方法を提供 |
| Modify (修正) | 地形の波の振幅や周期を変更する、複数のレーンを追加して難易度を上げる |
