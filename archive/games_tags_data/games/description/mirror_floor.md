# MIRROR FLOOR (mirror_floor)

**タグ**: #on_got_item-reverse_state, #field-bottomless, #on_pressed-jump, #field-auto_scroll

## 1. コア・メカニクス

| 項目 | 内容 |
|:---|:---|
| 入力 | 押す（ジャンプ）/ 長押し（加速） |
| 挙動 | 上下対称の鏡面世界を走り、コインを取得すると重力が反転し、反対側の世界に移動 |
| 終了条件 | 画面の上端または下端に到達するとゲームオーバー |
| スコア | コイン回収でmultiplier点（1, 2, 3...）、取り逃すとmultiplier-1 |
| 難易度 | スクロール速度が時間経過で増加し、ジャンプの精度がより要求される |

## 2. オブジェクト仕様

| 名前 | 形状 | 動作 | 衝突時 |
|:---|:---|:---|:---|
| player | char(a/b) | 床上を走行、Tapでジャンプ、Hold時に加速 | 画面上端/下端→終了 |
| floor | rect(幅45-75) | 自動スクロール、上下2面を持つ | プレイヤーが着地可能 |
| coin | char(c) | 床の上下に対になって表示、自動スクロール | +multiplier点、重力反転 |
| mirror_player | char(a/b) | プレイヤーの鏡像、反対側の世界に表示 | - |

## 3. デザインガイド分析

| 原則 | 評価 |
|:---|:---|
| (1) シンプルさと直感性 | 上下対称の鏡面構造で、2つの世界が視覚的に一体化。色の明暗（cyan/light_cyan, yellow/light_yellow）で現在の世界を明示 |
| (2) 視覚的フィードバック | ジャンプ時に「powerUp」音、着地時に「laser」音、コイン取得時に「coin」音。重力反転時に鏡像と入れ替わる視覚効果 |
| (3) リスク・リワード | コインを取るほどmultiplier増加だが、重力反転により予測困難な軌道に。取り逃すとmultiplier減少だが、安定した軌道を維持 |
| (4) 斬新なメカニクス | 単なる上下反転ではなく、鏡面世界を「行き来する」感覚。コインが報酬であると同時に強制的な世界切り替えトリガーとなる独特の緊張感 |

## 4. タグ詳細

| タグ | 役割 | 戦略的意義 |
|:---|:---|:---|
| #on_got_item-reverse_state | コイン取得時に重力反転、上下の世界が入れ替わる | コインを取るタイミングが次の着地点を左右。計画的なルート選択が必要 |
| #field-bottomless | 画面の上端/下端が即死ライン | ジャンプの高さを誤ると画面外に飛び出し死亡。重力反転後の軌道計算が重要 |
| #on_pressed-jump | Tapでジャンプ、重力に応じて上下どちらかに飛ぶ | ジャンプの高さと次の床の距離を見極める技術が求められる |
| #field-auto_scroll | 自動スクロールで床とコインが接近 | 反応速度が重要。Hold時の加速は高得点のチャンスだが、制御が難しくリスクも増大 |

## 5. 変奏ヒント

| 手法 | アイデア |
|:---|:---|
| Substitute (代用) | 上下反転を「左右反転」に変更し、画面の左右に鏡面世界を配置する横スクロール型に |
| Combine (結合) | 一部のコインは反転せず得点のみ、どのコインで反転するかを選択する戦略性を追加 |
| Modify (修正) | 3つ以上の平行世界を追加し、コインの色によって移動先の世界が決まる複雑な構造に |
