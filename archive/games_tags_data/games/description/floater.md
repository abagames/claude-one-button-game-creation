# FLOATER (floater)

**タグ**: field-floors, on_pressed-jump, field-bottomless

## 1. コア・メカニクス

| 項目 | 内容 |
|:---|:---|
| 入力 | タップでジャンプ |
| 長押し | 落下速度低減(0.05 vs 0.2の加速) |
| 挙動 | 浮遊床に着地中、タップでジャンプ。床の揺れに応じた初速が与えられる |
| 終了条件 | 画面下端または左端に到達 |
| スコア | 右への移動距離に応じて自動加算 |
| 難易度 | 自動スクロール速度増加、床の出現間隔・揺れ幅変化 |

## 2. オブジェクト仕様

| 名前 | 形状 | 動作 | 衝突時 |
|:---|:---|:---|:---|
| player | box | 重力で落下。長押しで落下減速。床上でタップでジャンプ | - |
| floater (青/水色) | box | 正弦波で上下に揺れながら左へスクロール | 着地。青は跳躍可能、水色は使用済み |
| ceiling | rect | 固定 | 接触時に速度減衰 |

## 3. デザインガイド分析

| 原則 | 評価 |
|:---|:---|
| (1) シンプルさと直感性 | プレイヤー、浮遊床、天井のみ。青は使用可能、水色は使用済みという色分け |
| (2) 視覚的フィードバック | 床の揺れが視覚的に予測可能。使用済み床の色変化で状態が明確 |
| (3) リスク・リワード | 床の揺れ上昇時にジャンプすれば高く跳べるが、タイミングミスで低ジャンプ。長押し滞空と早期着地の選択 |
| (4) 斬新なメカニクス | 「床の揺れ」が初速に影響する物理的連動。床の上昇タイミングでのジャンプが大ジャンプを生む |

## 4. タグ詳細

| タグ | 役割 | 戦略的意義 |
|:---|:---|:---|
| field-floors | 浮遊する床が主要な足場 | 床のタイミングと位置が全て。床の揺れ周期を読む習熟が重要 |
| on_pressed-jump | タップでジャンプ | タイミングゲーム。床の上昇時にジャンプすれば速度が加算され、高く跳べる |
| field-bottomless | 下端到達で即死 | 常に下からの圧力。床を逃すと即座に敗北という緊張感 |

## 5. 変奏ヒント

| 手法 | アイデア |
|:---|:---|
| Substitute (代用) | 床を「風の吹き出し口」に置き換え、接触時に上向きの力を受ける。揺れは風力の強弱に変化 |
| Combine (結合) | on_holding:thrustを追加し、長押しで水平推力も発生。着地位置の精密制御が可能に |
| Modify (修正) | 床に色を追加し、同色連続着地でボーナス。異色着地でコンボ切れ。戦略的な床選択 |
