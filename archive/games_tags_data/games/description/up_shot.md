# UP SHOT (up_shot)

**タグ**: #weapon-vertical, #on_holding-stop, #on_holding-shoot, #obstacle-fall, #field-auto_scroll

## 1. コア・メカニクス

| 項目 | 内容 |
|:---|:---|
| 入力 | 長押し：停止して上に連射 / 離す：走る（前進） |
| 挙動 | 画面下部を走るキャラを操作。長押しで停止して上に弾を撃ち、落ちてくる岩を破壊。離すと前進再開 |
| 終了条件 | 岩に接触、または画面左端に到達 |
| スコア | 岩に弾を当てると縮小してマルチプライヤー分のスコア + マルチプライヤー+1。完全破壊でマルチプライヤー×10点 |
| 難易度 | 時間経過で岩の落下速度と出現頻度が上昇 |

## 2. オブジェクト仕様

| 名前 | 形状 | 動作 | 衝突時 |
|:---|:---|:---|:---|
| player(a) | char("a") アニメ 黒色 | 画面下部を走行。離すと前進、長押しで停止して連射 | 岩と接触で終了 |
| shots | box(5,9) 黒色 | 長押し中に一定間隔で上方に発射。岩に当たると消失 | - |
| rocks | box 赤色 | 上から落下。サイズは5〜15でランダム。重力加速度で落下速度が増加 | 弾が当たるとサイズ×0.7に縮小。サイズ5未満で破壊 |
| ground | rect 薄黒/白 | 画面下部の地面。白線で区切られている | - |

## 3. デザインガイド分析

| 原則 | 評価 |
|:---|:---|
| (1) シンプルさと直感性 | 黒のプレイヤーと弾、赤い岩という明確な色分け。長押しで撃つ、離すと走るという二択の操作 |
| (2) 視覚的フィードバック | 射撃で「laser」音、岩縮小で「hit」音とパーティクル、岩破壊で「powerUp」音。前進再開で「select」音。状態が音で明確 |
| (3) リスク・リワード | 停止して撃つと安全に岩を破壊できるが、画面左端に押されるリスク。走ると安全だが岩を破壊できない |
| (4) 斬新なメカニクス | 走る/撃つの二択。停止してじっくり破壊するか、走って回避するかの判断が独特のリズムを生む |

## 4. タグ詳細

| タグ | 役割 | 戦略的意義 |
|:---|:---|:---|
| #weapon-vertical | 上方向に弾を連射 | 岩の落下軌道と弾の軌道を計算し、効率的に破壊する技術が必要。停止位置の選択が攻略の鍵 |
| #on_holding-stop | 長押しで停止し、その場に留まる | 停止のタイミングが生存とスコアを左右。岩の位置を見極めて最適な停止位置を選ぶ判断力が重要 |
| #on_holding-shoot | 長押しで連射を続ける | 押しっぱなしで連射し続けるため、どれだけ停止するかの時間管理が攻略の核心 |
| #obstacle-fall | 岩が上から落下して障害物となる | 落下する岩の軌道予測が必要。重力で加速するため、早めの対処が求められる |
| #field-auto_scroll | 画面が自動的に左にスクロールし続ける | 一定のペースで前進を強制され、停止しすぎると画面左端に到達。走る/撃つのバランスが重要 |

## 5. 変奏ヒント

| 手法 | アイデア |
|:---|:---|
| Substitute (代用) | 岩をミサイルや敵機に変更し、シューティングゲーム化。または上下を反転し、地面から岩が湧き出る設定に |
| Combine (結合) | アイテム収集要素を追加。破壊した岩からアイテムが落下し、取得で武器強化やスピードアップ |
| Modify (修正) | 左右移動も可能にし、岩の回避と射撃位置の選択を両立。または武器の種類を増やし、状況に応じた選択を要求 |
