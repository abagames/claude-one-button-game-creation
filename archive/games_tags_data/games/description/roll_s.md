# ROLL S (roll_s)

**タグ**: #on_holding-avoid, #on_holding-shoot, #on_pressed-turn, #field-auto_scroll, #field-lanes

## 1. コア・メカニクス

| 項目 | 内容 |
|:---|:---|
| 入力 | タップで角度変更 / 長押しで射撃 |
| 挙動 | プレイヤーは左端に固定。角度は5段階で自動往復。長押しで5方向に扇状射撃。射撃していない間は自動スクロール。 |
| 終了条件 | 敵または敵の弾に接触 |
| スコア | 敵を撃破で得点（マルチプライヤーあり） |
| 難易度 | 時間経過で敵の速度と射撃速度が増加 |

## 2. オブジェクト仕様

| 名前 | 形状 | 動作 | 衝突時 |
|:---|:---|:---|:---|
| player | カスタムキャラ | 左端固定、角度往復、長押しで5方向射撃 | - |
| shots | bar(3,3) | 射撃方向へ直進 | 敵を撃破 |
| enemies | カスタムキャラ | 右から出現、左へ移動、プレイヤーを狙って射撃 | 撃破で得点 |
| bullets | bar(3,3) | プレイヤーへ直進 | 終了 |

## 3. デザインガイド分析

| 原則 | 評価 |
|:---|:---|
| (1) シンプルさと直感性 | カスタムキャラクターと基本図形（bar）で構成。3つのレーンが緑色の線で明確に描画され、プレイエリアが一目で理解できる。 |
| (2) 視覚的フィードバック | 射撃時と敵撃破時に音とパーティクル。ゲームオーバーは敵または弾との衝突で明確。角度が視覚的にわかりやすい。 |
| (3) リスク・リワード | 射撃を続けると敵を倒せるがスクロールが止まる。射撃を止めるとスクロールが進んでマルチプライヤーが増加するが、敵に近づくリスクが高まる。高度な判断が求められる。 |
| (4) 斬新なメカニクス | 射撃と回避が同じボタンの長押し/離しで制御される点が独特。角度が自動的に往復し、タイミングを見計らって固定する仕組みも新鮮。 |

## 4. タグ詳細

| タグ | 役割 | 戦略的意義 |
|:---|:---|:---|
| #on_holding-shoot | 長押しで連続射撃、現在の角度に応じて5方向に扇状発射 | 角度を適切に固定してから射撃することで、複数の敵を同時に狙える戦略性が生まれる |
| #on_pressed-turn | タップで角度を固定・変更、5段階の角度を自動往復 | タイミングを見計らって角度を固定する技術が求められ、習熟によるスコア向上が見込める |
| #on_holding-avoid | 射撃を止める（ボタンを離す）とスクロールが進み、結果的に敵との距離が保たれる | 射撃と回避のバランスを取る必要があり、判断力が試される |
| #field-auto_scroll | 射撃していない間は自動スクロール、スクロール距離でマルチプライヤー増加 | スクロールを進めることでスコア倍率が上がるため、リスクを取って回避を選ぶインセンティブが生まれる |
| #field-lanes | 3つのレーンに敵が配置され、視覚的に明確に区分 | レーンごとに敵の配置を把握し、適切な角度で複数の敵を狙う戦略が可能 |

## 5. 変奏ヒント

| 手法 | アイデア |
|:---|:---|
| Substitute (代用) | 角度変更をテレポートに置き換え、3つのレーン間を瞬間移動する |
| Combine (結合) | 射撃と回避以外の第3のアクション（シールド展開など）を追加 |
| Modify (修正) | 角度の段階数を増やして8方向や16方向にし、より精密な狙いを可能にする |
