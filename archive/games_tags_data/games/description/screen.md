# SCREEN (screen)

**タグ**: #player-inverted, #on_pressed-shoot, #weapon-ray, #field-outpost

## 1. コア・メカニクス

| 項目 | 内容 |
|:---|:---|
| 入力 | タップで射撃 |
| 挙動 | プレイヤーは画面中央の線の両側を往復し、射撃可能な弾数（3発）がなくなると自動的に反対側へ移動して弾を補充。敵が上から降りてきて、画面下部に到達する前にレイを発射して撃破する。 |
| 終了条件 | 敵が画面下部（拠点）に到達 |
| スコア | 敵を撃破で得点（マルチプライヤーあり） |
| 難易度 | 時間経過でプレイヤーの回転速度と敵の降下速度が増加 |

## 2. オブジェクト仕様

| 名前 | 形状 | 動作 | 衝突時 |
|:---|:---|:---|:---|
| player | カスタムキャラ（a/b） | 中央線の両側を往復、3発撃つと反対側へ移動 | - |
| enemies | カスタムキャラ（c/d） | 上から降下 | レイに当たると得点 / 下部到達で終了 |
| shots | bar（青） | 射角方向へ直進 | 敵を撃破 |

## 3. デザインガイド分析

| 原則 | 評価 |
|:---|:---|
| (1) シンプルさと直感性 | 画面中央の線で左右が明確に区分され、プレイヤーの位置と弾数が一目瞭然。反対側の画面が暗くなり、現在のプレイエリアが明確。 |
| (2) 視覚的フィードバック | 射撃、敵撃破、サイド切り替えで異なる音。プレイヤーのレイ（bar）が視覚的に明確。弾数が画面下部にインジケーターで表示される。 |
| (3) リスク・リワード | 3発撃つと強制的に反対側へ移動するため、弾の使用タイミングが重要。撃ち逃すとマルチプライヤーが減少。サイド切り替え中は無防備。 |
| (4) 斬新なメカニクス | 画面を左右に分割し、プレイヤーが両側を往復する独特のメカニクス。レイの角度が自動的に回転し、タイミングを見計らって射撃する。 |

## 4. タグ詳細

| タグ | 役割 | 戦略的意義 |
|:---|:---|:---|
| #player-inverted | プレイヤーが画面中央の線の両側を往復、左右で鏡像 | 両サイドを管理し、弾数を意識して射撃する必要がある。サイド切り替えのタイミングが重要 |
| #on_pressed-shoot | タップで射撃、レイの角度は自動回転 | レイの角度を見計らって射撃するタイミング技術が求められる |
| #weapon-ray | レイ（bar）を発射、角度が自動回転 | 回転するレイの角度を予測し、適切なタイミングで射撃する必要がある |
| #field-outpost | 画面下部が拠点で、敵が到達するとゲームオーバー | 拠点防衛の緊張感が常にあり、敵を確実に撃破する必要がある |

## 5. 変奏ヒント

| 手法 | アイデア |
|:---|:---|
| Substitute (代用) | サイド切り替えを手動操作に変更し、プレイヤーが任意のタイミングで切り替えられるようにする |
| Combine (結合) | レイの角度を手動で調整できるようにし、より複雑な射撃戦略を可能にする |
| Modify (修正) | 弾数を増やして5-7発にし、より長くサイドに滞在できるようにする |
