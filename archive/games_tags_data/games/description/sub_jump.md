# SUB JUMP (sub_jump)

**タグ**: #field-water, #field-roughness, #on_pressed-jump, #field-auto_scroll

## 1. コア・メカニクス

| 項目 | 内容 |
|:---|:---|
| 入力 | 長押し |
| 挙動 | 潜水艦を操作し、水面より上（陸/空）と下（海）を行き来。ボタン長押しで上昇＋加速、離すと下降＋減速。地形に触れないようコインを取得 |
| 終了条件 | 地形（緑の陸または紫の海底）に衝突 |
| スコア | コイン取得で multiplier 点を獲得し multiplier +1。コインを取り逃すと multiplier -1 |
| 難易度 | 時間経過で加速度と移動速度が上昇。陸地の出現頻度が増加 |

## 2. オブジェクト仕様

| 名前 | 形状 | 動作 | 衝突時 |
|:---|:---|:---|:---|
| submarine | char(a/b) | ボタン長押しで上昇＋加速、離すと下降。水面より上では空気抵抗小、下では水抵抗大 | - |
| water | line(100幅,2) 青 | 画面中央を横断する水面（y=50）、越える時にパーティクル発生 | - |
| land | line(10間隔,2) 緑 | 水面上部の起伏のある地形、自動スクロール | ゲームオーバー |
| sea | line(10間隔,2) 紫 | 水面下部の起伏のある海底地形、自動スクロール | ゲームオーバー |
| coin | char(c) 黄色 | 地形付近にランダム出現、画面外に流れると消失 | +multiplier点、multiplier+1 |

## 3. デザインガイド分析

| 原則 | 評価 |
|:---|:---|
| (1) シンプルさと直感性 | 水面を境界とした上下の世界が明確。緑（陸）と紫（海）の色分けで危険エリアが一目瞭然。ボタン長押しで上昇という直感的な操作 |
| (2) 視覚的フィードバック | 水面突破時のジャンプ音とパーティクル、海中航行時の泡エフェクトが気持ちいい。潜水艦のアニメーション変化で状態を表現 |
| (3) リスク・リワード | コイン取得で multiplier が増加するが、取り逃すと減少。地形近くのコインほど危険だが、取れば高得点への道が開ける |
| (4) 斬新なメカニクス | 水面の上下で挙動が変わる物理特性。陸地と海底という異なる2種類の地形を同時に避ける独特の戦略性 |

## 4. タグ詳細

| タグ | 役割 | 戦略的意義 |
|:---|:---|:---|
| #field-water | 画面が水面を境に上（空中）と下（海中）に分かれる | 水中では抵抗が大きく慎重な操作が必要。空中では慣性が強く大胆な動きが可能。環境に応じた操作の切り替えが鍵 |
| #field-roughness | 陸地と海底が起伏のある不規則な地形を形成 | 固定パターンではなく動的に生成される地形により、先読みと即座の判断が求められる。海と陸で異なる起伏パターン |
| #on_pressed-jump | ボタン長押しで上昇（特に水中で効果的） | 単なるジャンプではなく、押し続けることで継続的に上昇。タイミングと持続時間の調整が重要 |
| #field-auto_scroll | 地形が自動で右から左へスクロール | プレイヤーの前進速度がボタン操作で変化。速く進めば難易度が上がるが、より多くのコインを取得可能 |

## 5. 変奏ヒント

| 手法 | アイデア |
|:---|:---|
| Substitute (代用) | 潜水艦を鳥に変え、雲（上）と地面（下）の間を飛行する空中アクションにする |
| Combine (結合) | 敵の魚や鳥を追加し、領域に応じて異なる敵が出現する回避要素を加える |
| Modify (修正) | 水面の高さ自体が波のように上下動し、安全地帯が常に変化する仕組みを追加 |
