# FOOT LASER (foot_laser)

**タグ**: on_pressed-jump, weapon-vertical, on_pressed-fall

## 1. コア・メカニクス

| 項目 | 内容 |
|:---|:---|
| 入力 | 押す（1回目: ジャンプ / 2回目: 2段ジャンプ / 3回目: 急降下） |
| 挙動 | 地面を走り、タップでジャンプ。空中では足から下方向に垂直レーザーが伸び、敵を倒すと倍率上昇 |
| 終了条件 | 敵に接触 |
| スコア | レーザーで敵を倒すと「倍率」点。倍率は撃破ごとに+1。地面着地で倍率1にリセット |
| 難易度 | 時間経過で敵の出現頻度増加、移動速度上昇 |

## 2. オブジェクト仕様

| 名前 | 形状 | 動作 | 衝突時 |
|:---|:---|:---|:---|
| player | char("a"/"b") 6x6 | 地面を走る。タップでジャンプ（最大2段）、3回目で急降下。重力で落下（長押しで落下速度減少） | 敵に接触: ゲームオーバー |
| laser（足レーザー） | rect(2, height) light_blue | 空中にいる間、足元から地面まで垂直に伸びる | 敵破壊時: +倍率点 |
| shots（足元弾） | rect(2, 9) purple | 空中にいる間、足元から自動発射される短い弾。地面に落ちると消滅 | - |
| enemy（地上） | char("c"/"d") 6x6 red | 左にスクロール。単体または壁（縦積み3〜6体） | レーザー接触: 破壊 |
| enemy（空中） | char("e"/"f") 6x6 red | 左にスクロール。高度50〜80に1〜5体の編隊で出現 | レーザー接触: 破壊 |

## 3. デザインガイド分析

| 原則 | 評価 |
|:---|:---|
| (1) シンプルさと直感性 | 「足から下に伸びるレーザー」という視覚的に明快なメカニクス。3回タップの挙動（ジャンプ→2段→急降下）がリズミカルで直感的 |
| (2) 視覚的フィードバック | 垂直レーザーの青色が常に表示され、攻撃範囲が明確。倍率がスコア表示時に視覚化。足元弾のパーティクルで地面着地を強調 |
| (3) リスク・リワード | 空中滞在時間が長いほど倍率が上がるが、地面着地で倍率リセット。急降下で危険回避か倍率リセットかの選択を迫られる |
| (4) 斬新なメカニクス | 「垂直レーザー」と「急降下」を組み合わせた独自システム。ジャンプ回数に応じて挙動が変わる3段階入力が戦略的深さを生む |

## 4. タグ詳細

| タグ | 役割 | 戦略的意義 |
|:---|:---|:---|
| on_pressed-jump | 1回目と2回目のタップでジャンプ・2段ジャンプ | 空中滞在時間を延ばし、レーザーで多くの敵を倒すための基本操作。タイミングと高度調整が習熟要素 |
| weapon-vertical | 空中で足元から地面まで垂直に伸びるレーザー | 下方向の敵を自動的に攻撃。プレイヤーは高度と位置を調整するだけで攻撃が成立。空中滞在が攻撃時間に直結 |
| on_pressed-fall | 3回目のタップで急降下 | 危険回避、着地タイミング調整、倍率リセット覚悟の緊急回避など、多様な戦術に使える。タップ回数管理が重要 |

## 5. 変奏ヒント

| 手法 | アイデア |
|:---|:---|
| Substitute (代用) | 垂直レーザーを「上方向レーザー」に変更。空中の敵を狙い、地面は安全地帯に |
| Combine (結合) | 倍率をリセットする代わりに「チャージ」システムに。着地でエネルギー回復、空中でエネルギー消費してレーザー発射 |
| Modify (修正) | 急降下にレーザー強化効果を追加。急降下中は太いレーザーになり、より多くの敵を倒せる |
