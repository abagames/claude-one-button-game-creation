# JUJUMP (jujump)

**タグ**: on_pressed-jump, on_pressed-turn, field-auto_scroll, field-floors, field-bottomless

## 1. コア・メカニクス

| 項目 | 内容 |
|:---|:---|
| 入力 | 押す |
| 挙動 | タップでジャンプと左右反転、空中ジャンプは威力が減衰（0.7倍） |
| 終了条件 | 画面下に落下 |
| スコア | 画面上部（y<30）に滞在すると得点、難易度に応じた基礎得点 |
| 難易度 | 時間経過で画面スクロール速度が増加 |

## 2. オブジェクト仕様

| 名前 | 形状 | 動作 | 衝突時 |
|:---|:---|:---|:---|
| player | 緑の四角（7×7） | タップで左右交互にジャンプ、重力で落下 | - |
| floor | 青の横バー（33×7） | 下から上へスクロール | 着地でジャンプ威力リセット |

## 3. デザインガイド分析

| 原則 | 評価 |
|:---|:---|
| (1) シンプルさと直感性 | 基本図形と色分け、左右交互のジャンプパターンが直感的 |
| (2) 視覚的フィードバック | ジャンプの放物線軌道、床着地時の位置調整で接触を明示 |
| (3) リスク・リワード | 空中ジャンプで危機回避できるが威力減衰、床着地でリセットのジレンマ |
| (4) 斬新なメカニクス | 左右交互ジャンプと威力減衰システム、床着地による威力回復 |

## 4. タグ詳細

| タグ | 役割 | 戦略的意義 |
|:---|:---|:---|
| on_pressed-jump | タップでジャンプ | 床着地と空中ジャンプのタイミング判断、威力管理が重要 |
| on_pressed-turn | タップで左右反転 | 左右交互のジャンプによる横移動制御、床の位置に合わせた調整 |
| field-auto_scroll | 画面が下から上へスクロール | 継続的な上昇を強制、床の流れに合わせたジャンプが必要 |
| field-floors | 複数の床が配置 | 床の間隔を見極め、空中ジャンプと床着地のバランス |
| field-bottomless | 画面下に落ちると終了 | 画面上部維持の緊張感、スクロール速度に対応する必要 |

## 5. 変奏ヒント

| 手法 | アイデア |
|:---|:---|
| Substitute (代用) | 左右反転を上下反転に変え、重力方向が反転するジャンプ |
| Combine (結合) | 床に色を追加し、同色の床でのみ威力リセット |
| Modify (修正) | タップ長押しでジャンプ威力をチャージ、威力減衰の代わりに使用 |
