# GOLFME (golfme)

**タグ**: on_holding-adjust_angle, on_released-throw, field-holes, field-bottomless, field-auto_scroll

## 1. コア・メカニクス

| 項目 | 内容 |
|:---|:---|
| 入力 | 長押し（角度調整） / 離す（ジャンプ） |
| 挙動 | 床の上を走り、長押しで角度調整、離すとジャンプ。穴を飛び越えて進む |
| 終了条件 | 画面左（x < 0）に落ちる、または画面下（y > 99）に落ちる |
| スコア | 自動的に距離に応じて加算（スクロール距離） |
| 難易度 | 時間経過でスクロール速度増加 |

## 2. オブジェクト仕様

| 名前 | 形状 | 動作 | 衝突時 |
|:---|:---|:---|:---|
| player | box(9,9) green | 床の上を走る。長押しで角度調整バー表示、離すとジャンプ | 床接触: 着地 |
| floor | rect(width,9) blue | 左にスクロール。穴（space）の間隔がランダム | - |
| hole（穴） | 空間 | 床と床の間の隙間。50〜150の可変幅 | 落下: ゲームオーバー |
| angle bar | bar(20,3) | 長押し中に表示される角度インジケーター。徐々に回転 | - |

## 3. デザインガイド分析

| 原則 | 評価 |
|:---|:---|
| (1) シンプルさと直感性 | ゴルフ風の「溜めて打つ」操作。角度バーで視覚的に分かりやすい。床と穴の色分けが明確 |
| (2) 視覚的フィードバック | 角度バーの回転、ジャンプの放物線、着地の瞬間。スコアが自動加算される様子 |
| (3) リスク・リワード | 長く溜めるほど遠くまで飛べるが、タイミングミスで画面外に。穴の幅がランダムで予測不能 |
| (4) 斬新なメカニクス | ゴルフ風の角度調整を横スクロールゲームに応用。自動スクロールとの組み合わせで、タイミングの重要性が増す |

## 4. タグ詳細

| タグ | 役割 | 戦略的意義 |
|:---|:---|:---|
| on_holding-adjust_angle | 長押しで角度バーが回転し、ジャンプ角度を調整 | 角度調整でジャンプ距離をコントロール。最適な角度を見極める習熟要素 |
| on_released-throw | ボタンを離すとその角度でジャンプ（投擲） | タイミングが重要。早すぎると短い、遅すぎると画面外に飛ぶ |
| field-holes | 床に穴が開いており、飛び越える必要がある | 穴の幅に応じてジャンプ距離を調整。予測と判断が戦略的 |
| field-bottomless | 画面下に落ちるとゲームオーバー | ジャンプミスは即死。緊張感を生む |
| field-auto_scroll | 自動的に左にスクロール | 常に前進を強要され、停滞できない。スクロール速度に応じた判断が必要 |

## 5. 変奏ヒント

| 手法 | アイデア |
|:---|:---|
| Substitute (代用) | 角度調整を「パワー調整」に変更。長押し時間でジャンプ力が変わる |
| Combine (結合) | 空中で方向キーを使って微調整できる「グライダー」機能を追加 |
| Modify (修正) | 床の高さがランダムに変化。上下の床を飛び移る立体的なゲームプレイ |
