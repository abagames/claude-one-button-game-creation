# UNCTRL (unctrl)

**タグ**: #rule-control_weapons, #rule-combo_multiplier, #on_holding-rotate

## 1. コア・メカニクス

| 項目 | 内容 |
|:---|:---|
| 入力 | タップ：ミサイル発射 / 長押し：ミサイルを上昇させる |
| 挙動 | 左側の青い戦車が固定位置にいる。タップで発射したミサイルを長押しで上下操作し、右から来る敵戦車と敵弾を破壊 |
| 終了条件 | 敵の弾に当たる |
| スコア | 敵戦車破壊でマルチプライヤー×10点。敵弾破壊でマルチプライヤー分のスコア獲得 |
| 難易度 | 時間経過で敵の移動速度と弾速が上昇し、出現頻度も増加 |

## 2. オブジェクト仕様

| 名前 | 形状 | 動作 | 衝突時 |
|:---|:---|:---|:---|
| player(a) | char("a") アニメ 青色 | 画面左側に固定。ミサイルを発射するのみ | 敵弾と接触で終了 |
| missile | bar(6,4) 青/薄青 | タップで発射。発射後は長押しで上昇、離すと下降。常に前進し続ける | - |
| enemy(a) | char("a") アニメ 赤色 | 右から左に移動。画面中央より右では弾を発射 | ミサイルと接触でマルチプライヤー×10点 |
| enemy_bullets | bar(4,2) 赤色 | 敵戦車からランダムな角度で発射。プレイヤーに向かって飛来 | ミサイルで破壊でマルチプライヤー分のスコア + マルチプライヤー×2 |

## 3. デザインガイド分析

| 原則 | 評価 |
|:---|:---|
| (1) シンプルさと直感性 | 青の味方、赤の敵という明確な色分け。長押しで上、離すと下という直感的な操作 |
| (2) 視覚的フィードバック | 発射で「select」音、上昇で「hit」音、下降で「laser」音。敵破壊で「powerUp」音とパーティクル。敵弾破壊で「coin」音。状態が明確 |
| (3) リスク・リワード | 敵弾を破壊するとマルチプライヤーが2倍になるが、失敗すると被弾の危険。高リスク高リターン |
| (4) 斬新なメカニクス | ミサイル発射後に操作する独特のシステム。一度に1発しか制御できず、次の弾のためには敵を倒すか外すかの選択 |

## 4. タグ詳細

| タグ | 役割 | 戦略的意義 |
|:---|:---|:---|
| #rule-control_weapons | 発射後のミサイルを操作して軌道を制御 | ミサイルの軌道計算と敵の動きの予測が攻略の核心。発射タイミングと軌道修正の技術が必要 |
| #rule-combo_multiplier | 敵弾破壊で倍々にマルチプライヤー増加（最大64倍） | 敵を倒すか敵弾を破壊するかの選択。コンボ維持で高スコアだが失敗のリスクも増大 |
| #on_holding-rotate | 長押しで上昇、離すと下降してミサイルの軌道を変更 | タイミング調整が生存とスコアを左右。押しっぱなしではなく、適切なタイミングで離す判断力が重要 |

## 5. 変奏ヒント

| 手法 | アイデア |
|:---|:---|
| Substitute (代用) | ミサイルを誘導レーザーに変更し、連続的なビームで敵を破壊。または複数のミサイルを同時発射可能に |
| Combine (結合) | アイテム収集要素を追加。敵破壊時にパワーアップアイテムが出現し、ミサイル速度や操作性が向上 |
| Modify (修正) | プレイヤーも左右に移動可能に。ミサイル制御と自機の位置調整の両立を要求し、難易度を上げる |
