# D PISTOLS (d_pistols)

**タグ**: on_pressed-shoot, on_holding-shoot, on_pressed-turn

## 1. コア・メカニクス

| 項目 | 内容 |
|:---|:---|
| 入力 | タップで方向転換&発射 / 長押しで十字発射 |
| 挙動 | タップ時は進行方向の前後2方向へ発射。30フレーム長押しで十字4方向へ発射 |
| 終了条件 | 敵がプレイヤーに接触 |
| スコア | 敵撃破時にmultiplier値を獲得。攻撃ごとにmultiplierは減少 |
| 難易度 | プレイヤーの移動速度上昇、敵の出現頻度・速度増加、長押しチャージ速度上昇 |

## 2. オブジェクト仕様

| 名前 | 形状 | 動作 | 衝突時 |
|:---|:---|:---|:---|
| player | char(a,b) | 自動で上下に移動。タップで方向転換。長押し中は減速 | 敵接触でゲームオーバー |
| shots | bar(1,6) | 発射方向へ直進 | - |
| enemy | char(c,d) | 画面左右から出現し、中央で方向転換してプレイヤーを追尾 | +multiplier点 |

## 3. デザインガイド分析

| 原則 | 評価 |
|:---|:---|
| (1) シンプルさと直感性 | ピクセルアートキャラと弾のみ。タップで方向転換、長押しで強力な攻撃という直感的な2段階操作 |
| (2) 視覚的フィードバック | 敵が中央で方向転換して追尾する動きが明確。長押しチャージは30フレームの待機で視覚化される |
| (3) リスク・リワード | 強力な十字発射はmultiplierを5減らし、移動も止まるリスク。通常攻撃は1減少で軽微 |
| (4) 斬新なメカニクス | 「方向転換と発射が同時」「長押しで十字発射」という2つの攻撃モードを持つデュアルピストル。敵の追尾挙動との駆け引きが独特 |

## 4. タグ詳細

| タグ | 役割 | 戦略的意義 |
|:---|:---|:---|
| on_pressed-shoot | タップで前後2方向へ発射 | 方向転換と攻撃を同時に行う独特の操作感。タップのタイミングが生存と得点の両方に影響 |
| on_holding-shoot | 30フレーム長押しで十字4方向へ発射 | 広範囲カバーだが移動停止とmultiplier大幅減少のリスク。緊急時の強力な選択肢 |
| on_pressed-turn | タップで移動方向を反転 | 限定された空間での移動制御。攻撃と移動が連動し、意図しない方向転換のリスクを生む |

## 5. 変奏ヒント

| 手法 | アイデア |
|:---|:---|
| Substitute (代用) | 長押し攻撃を「時間停止フィールド」に置き換え、フィールド内の敵を一掃。時間経過でmultiplier減少 |
| Combine (結合) | weapon:auto_aimingを追加し、通常攻撃は前後、長押しは最も近い敵4体への自動追尾弾を発射 |
| Modify (修正) | 長押し時間に応じて発射方向数が増加(4方向→8方向→16方向)。より細かいコストとリターンの調整 |
