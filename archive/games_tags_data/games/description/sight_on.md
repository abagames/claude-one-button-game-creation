# SIGHT ON (sight_on)

**タグ**: #weapon-auto_aiming, #weapon-explosion, #field-outpost, #on_pressed-shoot

## 1. コア・メカニクス

| 項目 | 内容 |
|:---|:---|
| 入力 | タップで射撃 |
| 挙動 | 照準が自動的に最も低い位置の敵を追尾。敵はターゲットに向かって飛行し、画面下部に到達しようとする。タップすると照準位置に爆発を発生させ、範囲内の敵を撃破。タップごとにマルチプライヤーがリセットされ、ターゲットが下方向に移動。 |
| 終了条件 | 敵が画面下部（拠点）に到達 |
| スコア | 敵を撃破で得点（マルチプライヤーあり） |
| 難易度 | 時間経過で敵の速度と追尾性能が増加 |

## 2. オブジェクト仕様

| 名前 | 形状 | 動作 | 衝突時 |
|:---|:---|:---|:---|
| enemies | カスタムキャラ（a） | ターゲットに向かって飛行、自動追尾 | 爆発範囲内で撃破 / 下部到達で終了 |
| targets | 移動するターゲット | ランダムに移動、敵の目標地点 | - |
| sight | カスタムキャラ（b） | 最も低い位置の敵を自動追尾 | - |
| explosions | box（黄） | タップ時に照準位置で発生 | 範囲内の敵を撃破 |

## 3. デザインガイド分析

| 原則 | 評価 |
|:---|:---|
| (1) シンプルさと直感性 | 照準（sight）が敵を自動追尾し、タップで爆発。敵がターゲットに向かう様子が視覚的に明確。星のアニメーションが美しく、宇宙空間の雰囲気を演出。 |
| (2) 視覚的フィードバック | 爆発エフェクト、敵撃破の音とパーティクル、ゲームオーバーの音。照準の移動が滑らかで、追尾している様子が一目瞭然。 |
| (3) リスク・リワード | タップすると爆発で敵を撃破できるが、マルチプライヤーがリセットされる。タップのタイミングを見計らって、複数の敵を同時に撃破することで高得点を狙える。ターゲットが下方向に移動するペナルティもある。 |
| (4) 斬新なメカニクス | 照準が自動追尾するため、プレイヤーはタイミングだけを考える。敵が誘導ミサイルのようにターゲットを追尾する独特のメカニクス。爆発範囲で複数の敵を同時撃破できる戦略性。 |

## 4. タグ詳細

| タグ | 役割 | 戦略的意義 |
|:---|:---|:---|
| #weapon-auto_aiming | 照準が最も低い位置の敵を自動追尾 | プレイヤーは照準を操作せず、射撃のタイミングだけに集中できる。単純化されたコントロール |
| #weapon-explosion | タップで照準位置に爆発、範囲内の敵を撃破 | 爆発範囲を利用して、複数の敵を同時撃破する戦略が可能 |
| #field-outpost | 画面下部が拠点で、敵が到達するとゲームオーバー | 拠点防衛の緊張感が常にあり、最も危険な敵を優先して撃破する必要がある |
| #on_pressed-shoot | タップで爆発を発生、マルチプライヤーがリセットされる | タップのタイミングが重要。連続撃破を狙ってマルチプライヤーを上げてから爆発させる戦略が求められる |

## 5. 変奏ヒント

| 手法 | アイデア |
|:---|:---|
| Substitute (代用) | 自動照準を手動照準に変更し、プレイヤーが照準を操作できるようにする |
| Combine (結合) | 複数の照準を同時に配置し、より複雑な戦略を可能にする |
| Modify (修正) | 爆発範囲を調整し、より精密な射撃を求めるようにする |
