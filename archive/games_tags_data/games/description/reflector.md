# REFLECTOR (reflector)

**タグ**: #weapon-explosion, #weapon-interference, #player-reciprocate, #on_holding-defenseless, #weapon-vertical

## 1. コア・メカニクス

| 項目 | 内容 |
|:---|:---|
| 入力 | タップで方向転換、長押しで反射板強化（無防備になる） |
| 挙動 | UFOが左右に往復し、反射板で敵弾を跳ね返して戦車を破壊、連鎖で高得点 |
| 終了条件 | UFOが敵弾に被弾 |
| スコア | 反射弾で戦車を破壊すると multiplier × 1点、連鎖で multiplier 増加 |
| 難易度 | UFOの速度と戦車の出現頻度・発射速度が上昇 |

## 2. オブジェクト仕様

| 名前 | 形状 | 動作 | 衝突時 |
|:---|:---|:---|:---|
| ufo | char("a", "b") | 左右に往復、長押しで減速 | 敵弾で終了 |
| reflector | bar(9, 3) | UFOの前方に配置、長押しで強化（短く太く） | 敵弾を反射・加速 |
| tank | char("c", "d") | 左右から出現、UFOを狙って砲撃 | 反射弾で破壊→得点 |
| bullet | bar(4, 3) | 戦車から発射（紫）、反射後は赤に変化 | 反射板で跳ね返る |
| explosion | arc(radius) | 反射弾が戦車/地面に着弾時に発生 | 戦車を破壊 |

## 3. デザインガイド分析

| 原則 | 評価 |
|:---|:---|
| (1) シンプルさと直感性 | UFO、反射板、戦車、弾という明快な要素。青い反射板と紫→赤に変化する弾の色分けが明確。 |
| (2) 視覚的フィードバック | 弾の反射と加速、爆発の広がるアニメーション、反射板の強化による視覚的変化。 |
| (3) リスク・リワード | 長押しで反射板を強化すると弾を強く跳ね返せるが、UFOが減速し無防備になる。連鎖破壊による高得点。 |
| (4) 斬新なメカニクス | 敵弾を反射して攻撃に転用する防御と攻撃の一体化。反射の角度と威力を強化できる可変反射板。 |

## 4. タグ詳細

| タグ | 役割 | 戦略的意義 |
|:---|:---|:---|
| #weapon-explosion | 反射弾が着弾すると爆発し、範囲内の戦車を破壊 | 複数の敵を同時に破壊する戦略性と爽快感を生む |
| #weapon-interference | 敵弾を反射して攻撃に転用 | 敵の攻撃を防ぐだけでなく、逆に利用する戦略的深みを提供 |
| #player-reciprocate | UFOが左右に往復移動 | 予測可能な移動パターンにより反射のタイミングに集中できる |
| #on_holding-defenseless | 長押しで反射板強化するが、UFOが減速し無防備に | リスクとリワードのトレードオフを生み出し、戦略的な使い分けを促す |
| #weapon-vertical | 反射弾が縦方向にも飛ぶ | 反射の角度により様々な方向に攻撃でき、戦略的な反射位置の選択を可能にする |

## 5. 変奏ヒント

| 手法 | アイデア |
|:---|:---|
| Substitute (代用) | UFOを宇宙船に、反射板をシールドに変更してSFテーマに |
| Combine (結合) | 複数の反射板を異なる角度で配置し、複雑な反射パターンを実現 |
| Modify (修正) | 長押し時間で反射板のサイズが変化し、より大きな範囲を防御可能に |
