# S LANES (s_lanes)

**タグ**: #field-lanes, #field-outpost, #on_holding-shoot, #on_holding-move

## 1. コア・メカニクス

| 項目 | 内容 |
|:---|:---|
| 入力 | 長押しで射撃と前進 |
| 挙動 | プレイヤーは4つのレーンを移動。長押しで射撃しながら前進、離すとレーンを切り替え。敵を撃破するとコインが出現し、取るとマルチプライヤーが増加。取り逃すとマルチプライヤー減少とレーン速度増加。 |
| 終了条件 | 敵が左端（拠点）に到達 |
| スコア | コインを取得で得点（マルチプライヤーあり） |
| 難易度 | 時間経過で敵の速度が増加 |

## 2. オブジェクト仕様

| 名前 | 形状 | 動作 | 衝突時 |
|:---|:---|:---|:---|
| ship | カスタムキャラ（a） | 長押しで前進と射撃、離すとレーン切り替え | - |
| enemies | カスタムキャラ（b） | 右から左へ移動、レーン速度に影響される | 撃破されるとコインになる |
| coins | カスタムキャラ（c） | 左へ移動 | 取得でマルチプライヤー+1 / 取り逃すとマルチプライヤー-1とレーン速度増加 |
| shots | カスタムキャラ（d） | 右へ直進 | 敵を撃破 |

## 3. デザインガイド分析

| 原則 | 評価 |
|:---|:---|
| (1) シンプルさと直感性 | カスタムキャラクターで視覚的に明確。4つのレーンが明確に区分され、「<」記号でレーン速度が視覚化されている。 |
| (2) 視覚的フィードバック | 射撃、敵撃破、コイン取得、コイン取り逃しで異なる音とエフェクト。ゲームオーバーは敵の到達地点に「X」マークが表示され、失敗の理由が一目瞭然。 |
| (3) リスク・リワード | 長押しで前進して射撃するが、レーン切り替えができない。コインを取り逃すとマルチプライヤーが減少し、レーン速度が増加する重大なペナルティ。攻撃と防御のバランスが求められる。 |
| (4) 斬新なメカニクス | 長押しで射撃と前進、離すとレーン切り替えという1入力2モードの設計が独特。コインを取り逃すとレーン速度が増加する独特のペナルティシステムが緊張感を生む。 |

## 4. タグ詳細

| タグ | 役割 | 戦略的意義 |
|:---|:---|:---|
| #field-lanes | 4つのレーンでプレイエリアを区分、各レーンは独立した速度を持つ | レーンごとの速度を管理し、危険なレーンを避けつつコインを取る戦略が必要 |
| #field-outpost | 左端が拠点で、敵が到達するとゲームオーバー | 拠点防衛の緊張感が常にあり、敵を確実に撃破する必要がある |
| #on_holding-shoot | 長押しで連続射撃、射撃中は前進も行う | 射撃のタイミングと位置取りが重要。前進しすぎると危険なレーンに入る |
| #on_holding-move | 長押しで前進、射撃位置を調整できる | コインを取るために前進する必要があるが、前進しすぎると敵に近づきすぎる |

## 5. 変奏ヒント

| 手法 | アイデア |
|:---|:---|
| Substitute (代用) | レーン切り替えをテレポートに置き換え、瞬間的に別のレーンへ移動 |
| Combine (結合) | 射撃と移動を独立した操作に分離し、より複雑な戦略を可能にする |
| Modify (修正) | レーン数を6-8に増やし、複数のレーンを同時に管理する難易度を上げる |
