# CYWALL (cywall)

**タグ**: on_pressed-jump_into_another

## 1. コア・メカニクス

| 項目 | 内容 |
|:---|:---|
| 入力 | 押す |
| 挙動 | 円の中心点から次の円へジャンプ、経路上に壁があればゲームオーバー |
| 終了条件 | 壁に衝突、またはプレイヤーの円が画面下（y>99）に落ちる |
| スコア | スクロール距離に応じた得点（0.1×difficulty + 位置補正） |
| 難易度 | 時間経過でスクロール速度増加、壁の回転速度増加（0.02-0.05×difficulty） |

## 2. オブジェクト仕様

| 名前 | 形状 | 動作 | 衝突時 |
|:---|:---|:---|:---|
| player | box(5,5) 青 / box(3,3) 青 | 円の中心点、次の円へジャンプ移動 | - |
| circle | box(3,3) 赤/青 | 半径20-30、上から下へスクロール、1-4本の壁を持つ | - |
| wall | bar(10-20, 3) 赤 | 円周上を回転（0.02-0.05×difficulty）、1-4本配置 | ジャンプ経路で衝突 |
| path | line(3) 青 | プレイヤーから次の円への経路、壁判定に使用 | 壁との衝突判定 |

## 3. デザインガイド分析

| 原則 | 評価 |
|:---|:---|
| (1) シンプルさと直感性 | 円から円へのジャンプという直感的な動き、色分け（青=安全、赤=危険） |
| (2) 視覚的フィードバック | 壁の回転アニメーション、ジャンプ時のパーティクル効果、プレイヤーと次の円の色変化 |
| (3) リスク・リワード | 壁の回転を読んでタイミングを計る、早めのジャンプと安全な待機の選択 |
| (4) 斬新なメカニクス | 円から円へのジャンプ（jump_into_another）、経路上の壁判定、回転する障害物パターン |

## 4. タグ詳細

| タグ | 役割 | 戦略的意義 |
|:---|:---|:---|
| on_pressed-jump_into_another | 押すと次の円へジャンプ、経路上に壁があればゲームオーバー、パーティクル効果 | タイミングの精密制御、壁の回転パターン読み、複数の壁の隙間を狙う習熟 |

## 5. 変奏ヒント

| 手法 | アイデア |
|:---|:---|
| Substitute (代用) | 円を他の図形に変更 / 壁を別の障害物（敵、レーザーなど）に変更 |
| Combine (結合) | 複数のジャンプ先を選択可能に / 壁の種類を増やして特殊効果を追加 |
| Modify (修正) | 壁の数を増やして難易度アップ / 回転速度をより予測可能に / ジャンプ時の無敵時間を追加 |
