# FROOOOG (froooog)

**タグ**: on_holding-adjust_angle, on_released-throw, field-auto_scroll

## 1. コア・メカニクス

| 項目 | 内容 |
|:---|:---|
| 入力 | 長押し（しゃがんでジャンプ距離を調整） / 離す（ジャンプ） |
| 挙動 | カエルが下から上にジャンプ。長押しで距離を調整し、離すとジャンプ。車を避けながら上を目指す |
| 終了条件 | 車に接触、または画面下（y > 103）に落下 |
| スコア | (ジャンプ距離 / 10)² 点。安全地帯（緑帯）着地は0点だが安全 |
| 難易度 | 時間経過とカエルの高さに応じてスクロール速度増加 |

## 2. オブジェクト仕様

## 3. デザインガイド分析

| 原則 | 評価 |
|:---|:---|
| (1) シンプルさと直感性 | 長押しで「溜め」、離して「ジャンプ」という直感的な操作。ターゲットインジケーターで着地点が明確。安全地帯の緑色が視覚的に分かりやすい |
| (2) 視覚的フィードバック | しゃがみアニメーション、ジャンプの放物線、着地時のパーティクル不在（シンプル）。車の色でレーンを区別 |
| (3) リスク・リワード | 長いジャンプほど高得点だが車との接触リスク増。安全地帯は0点だが確実に生き残れる。得点が2乗なので長距離ジャンプの価値が高い |
| (4) 斬新なメカニクス | Frogger風だが、長押しで距離調整する独自システム。ジャンプ中の放物線スケーリング演出が視覚的魅力。スクロール速度が高さに応じて増加し、上に行くほど難しくなる |

## 4. タグ詳細

| タグ | 役割 | 戦略的意義 |
|:---|:---|:---|
| on_holding-adjust_angle | 長押しでしゃがみ、ターゲット位置が上方向に伸びる | ジャンプ距離を精密にコントロール。長押し時間と車の動きを見極める習熟要素。最適なタイミングでのリリースが高得点の鍵 |
| on_released-throw | ボタンを離すとジャンプ（投擲のような挙動） | タイミング判断が重要。車の隙間を見極め、リリースのタイミングで成否が決まる。緊張感のある瞬間的判断 |
| field-auto_scroll | フィールドが下から上へ自動スクロール | 常に前進を強要され、停滞できない。カエルの高さに応じて加速し、プレッシャーが増大 |

## 5. 変奏ヒント

| 手法 | アイデア |
|:---|:---|
| Substitute (代用) | 車を「流れる丸太」に変更。丸太の上に着地すると一緒に移動し、次のジャンプのタイミングが重要に |
| Combine (結合) | ジャンプ中に左右移動できる機能を追加。軌道を微調整して車を避ける |
| Modify (修正) | 長押し時間に上限を設け、「溜めすぎると自動ジャンプ」。リスク管理の新たな要素 |
