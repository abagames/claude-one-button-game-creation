# FLIPBOMB (flipbomb)

**タグ**: on_pressed-reverse_state, weapon-explosion, rule-chain, rule-physics, field-outpost

## 1. コア・メカニクス

| 項目 | 内容 |
|:---|:---|
| 入力 | タップでフリッパー反転 |
| 挙動 | ボールがフリッパーで跳ね上がり爆弾を誘爆。爆発の連鎖反応を狙う |
| 終了条件 | 爆弾が地面またはフリッパーに接触 |
| スコア | 爆弾を誘爆させた際のmultiplier値。タップでリセット |
| 難易度 | ボール・爆弾の出現頻度と速度上昇 |

## 2. オブジェクト仕様

| 名前 | 形状 | 動作 | 衝突時 |
|:---|:---|:---|:---|
| flipper (青) | line×2 | タップで角度反転。ボールを跳ね上げる | 爆弾接触でゲームオーバー |
| ball (青/水色) | box | フリッパーから落下。タップで跳ね上げて投射 | - |
| bomb (赤) | box | 放物線で降下 | ボール/爆発で誘爆、地面で敗北 |
| explosion (紫) | box×5 | 爆弾誘爆時に回転する5つの爆発エリア | 爆弾を連鎖誘爆 |

## 3. デザインガイド分析

| 原則 | 評価 |
|:---|:---|
| (1) シンプルさと直感性 | フリッパー、ボール、爆弾の3要素のみ。タップでフリッパー反転という単純操作 |
| (2) 視覚的フィードバック | 爆発の回転する星型パターンが印象的。ボールの軌跡パーティクルで弾道が視覚化 |
| (3) リスク・リワード | タップは強力な跳ね上げだがmultiplierリセット。タイミングを待つか即座に反撃するか |
| (4) 斬新なメカニクス | フリッパーが防御と攻撃を兼ねる。爆発の連鎖反応による雪崩式の撃破が爽快 |

## 4. タグ詳細

| タグ | 役割 | 戦略的意義 |
|:---|:---|:---|
| on_pressed-reverse_state | フリッパーの角度を反転 | 単純な2状態切り替えだが、ボールの跳ね上げ角度に大きく影響。タイミングが全て |
| weapon-explosion | 爆弾誘爆で範囲攻撃 | 点ではなく面での攻撃。爆発範囲の理解と連鎖の計画が重要 |
| rule-chain | 爆発が他の爆弾を誘爆 | 1つの爆発から連鎖反応。初手の精度が連鎖の規模を決定 |
| rule-physics | 放物線運動の物理計算 | ボールと爆弾の軌道予測。フリッパー角度と初速の関係を体得する必要 |
| field-outpost | フリッパー(地面)を守る | 爆弾の地面到達が敗北条件。防衛ラインの明確な緊張感 |

## 5. 変奏ヒント

| 手法 | アイデア |
|:---|:---|
| Substitute (代用) | 爆発を「時間停止フィールド」に置き換え、範囲内の爆弾を一時停止。連鎖は時間差で発動 |
| Combine (結合) | rule:combo_multiplierを維持し、連鎖数でmultiplierが上昇。タップでリセットされる緊張感を強化 |
| Modify (修正) | 爆弾に色を追加し、同色連鎖でボーナス。異色は連鎖せず、戦略的な標的選択が必要に |
