# PIZZA ARROW (pizza_arrow)

**タグ**: #on_released-throw, #rule-time_manipulation, #rule-time_limit

## 1. コア・メカニクス

| 項目 | 内容 |
|:---|:---|
| 入力 | 長押し（時間減速）+ 離す（矢を放つ） |
| 挙動 | 回転するピザに矢を放ち、外側の黄色部分を切り取る |
| 終了条件 | 矢が中心の赤い部分（core）に衝突 |
| スコア | 切り取ったピザの角度 × 100 × multiplier（連続成功で上昇） |
| 難易度 | ピザの回転速度が一定、矢の本数制限（初期1本、全て成功で+1本） |

## 2. オブジェクト仕様

| 名前 | 形状 | 動作 | 衝突時 |
|:---|:---|:---|:---|
| pizza | arc(20, 4, 扇形) | 一定速度で回転、黄色の外周と赤い中心 | - |
| arrow | char("a") | 長押しで減速、離すと左へ高速移動 | 黄色→切り取り+得点、赤→終了 |
| pizzaPart | arc(20, 4, 扇形) | 切り取られたピザが左下へ落下 | - |

## 3. デザインガイド分析

| 原則 | 評価 |
|:---|:---|
| (1) シンプルさと直感性 | ピザと矢という明快なモチーフ。黄色の食べられる部分と赤い危険な中心が色で区別され、ルールが視覚的に自明。 |
| (2) 視覚的フィードバック | 切り取られたピザが飛んでいくアニメーション、長押し時の時間減速エフェクト、矢の発射時のライン表示で狙いが明確。 |
| (3) リスク・リワード | 中心に近い部分を狙うほど高得点だが赤い部分に当たるリスク増。連続成功でmultiplier上昇により習熟が報酬に直結。時間減速の戦略的使用。 |
| (4) 斬新なメカニクス | 時間操作をチャージではなく「狙いの精度向上」に使う逆転の発想。矢の本数制限と回転ピザの組み合わせによる緊張感の演出。 |

## 4. タグ詳細

| タグ | 役割 | 戦略的意義 |
|:---|:---|:---|
| #on_released-throw | 長押しで時間を遅くし、離すことで矢を放つ | タイミングの精密なコントロールを可能にし、スキルベースのプレイを実現 |
| #rule-time_manipulation | 長押し中はゲーム速度が0.05倍に減速 | 難しい状況でも落ち着いて狙える救済措置であり、戦略的な使い分けを促す |
| #rule-time_limit | 矢の本数が制限され、全て成功して初めて1本追加 | 完璧なプレイを要求することで緊張感と達成感を生み出す |

## 5. 変奏ヒント

| 手法 | アイデア |
|:---|:---|
| Substitute (代用) | ピザを惑星や時計盤に変更し、テーマを宇宙やタイムトラベルに |
| Combine (結合) | 複数のピザが異なる速度で回転し、全てを切り取る必要がある |
| Modify (修正) | 長押し時間に応じて矢の威力が変化し、大きな部分を一度に切り取れる |
